# <a name="change-data-capture-for-oracle-by-attunity"></a>For Oracle by Attunity의 변경 데이터 캡처
Oracle by Attunity 다운로드에 대 한 변경 데이터 캡처는 다음과 같은 2 구성 요소에 포함 됩니다.

-   [Change Data Capture Designer for Oracle by Attunity](../../integration-services/change-data-capture/change-data-capture-designer-for-oracle-by-attunity.md)

-   [Change Data Capture Service for Oracle by Attunity](../../integration-services/change-data-capture/change-data-capture-service-for-oracle-by-attunity.md)   

## <a name="get-the-change-data-capture-for-oracle-by-attunity-download"></a>Attunity 다운로드를 통해 Oracle에 대 한 변경 데이터 캡처을 가져오기
Microsoft SQL Server® 2016용 Microsoft® Change Data Capture Designer and Service for Oracle by Attunity는 SQL Server 2016 기능 팩의 일부입니다. [SQL Server 2016 기능 팩 웹 페이지](http://go.microsoft.com/fwlink/?LinkId=746297)에서 기능 팩의 구성 요소를 다운로드합니다.
