# 원본 길잡이
  원본 길잡이 구성 요소는 원본 구성 요소 및 연결 관리자 만들기를 도와줍니다. 이 구성 요소는 SSIS 도구 상자의 **즐겨찾기** 섹션에 있습니다.  
  
> [!NOTE]  
>  원본 길잡이는 Integration Services 연결 프로젝트 및 해당 마법사를 대체합니다.  
  
## 관련 작업  
 원본 길잡이를 사용하여 원본 및 연결 관리자를 추가하는 방법은 [Add a Source using Source Assistant](../Topic/Add%20a%20Source%20using%20Source%20Assistant.md)를 참조하십시오.
