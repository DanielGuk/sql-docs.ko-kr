# <a name="raw-file-source-editor-columns-page"></a>원시 파일 원본 편집기(열 페이지)
  원시 파일 원본은 파일에서 원시 데이터를 읽습니다. 원본의 기본 데이터 표현을 사용하므로 데이터를 변환하거나 거의 구문 분석할 필요도 없습니다. 자세한 내용은 [Raw File Source](../../integration-services/data-flow/raw-file-source.md)을 참조하세요.
