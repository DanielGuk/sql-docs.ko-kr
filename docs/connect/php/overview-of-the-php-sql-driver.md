# <a name="overview-of-the-microsoft-drivers-for-php-for-sql-server"></a>Microsoft Drivers for PHP for SQL Server의 개요

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

[!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] 는 SQL Server 2005 이상에 Azure SQL 데이터베이스를 포함 하 여 데이터 액세스를 제공 하는 PHP 확장입니다. 확장의 SQL Server Express를 SQL Server 2005부터 포함 한 모든 버전의 데이터에 액세스 하기 위해 SQLSRV 드라이버은 단계별 인터페이스 및 PDO_SQLSRV 드라이버를 사용 하 여 개체 지향 인터페이스를 제공 합니다. 버전 3.1 및 드라이버의 나중에 대 한 지원이 SQL Server 2008로 시작합니다. [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] API에는 Windows 인증, 트랜잭션, 매개 변수 바인딩, 스트리밍, 메타데이터 액세스 및 오류 처리에 대한 지원이 포함됩니다.  
  
사용 하는 [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)]올바른 버전 SQL Server Native Client의 경우, 또는 Microsoft ODBC Driver PHP 동일한 컴퓨터에 설치 되어 실행 되 고 있습니다.  자세한 내용은 참조 [Microsoft Drivers for PHP for SQL Server에 대 한 시스템 요구 사항](../../connect/php/system-requirements-for-the-php-sql-driver.md)합니다.
