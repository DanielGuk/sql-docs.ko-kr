# <a name="python-sql-driver"></a>Python SQL 드라이버

![다운로드-아래쪽 화살표-원이](../../ssdt/media/download.png)[for Python SQL 드라이버를 설치 하려면](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Windows, Linux 또는 Mac.에서 Python을 사용 하 여 SQL Database에 연결할 수 있습니다.   
  
## <a name="getting-started"></a>시작  
몇 가지 python SQL 드라이버는 사용할 수 있습니다. Microsoft 테스트 노력 및에 대 한 신뢰성을 배치 하는 반면 **pyodbc** 드라이버입니다. 드라이버를 선택 하 고 개발 환경을 적절 하 게 구성 합니다.
* [Python SQL 드라이버 - pyodbc](pyodbc/python-sql-driver-pyodbc.md)
* [Python SQL 드라이버 - pymssql](pymssql/python-sql-driver-pymssql.md)
