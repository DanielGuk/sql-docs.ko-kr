# <a name="download-microsoft-jdbc-driver-for-sql-server"></a>Microsoft JDBC Driver for SQL Server 다운로드

![아래쪽 화살표-Circled 다운로드](../../ssdt/media/download.png)[JDBC 드라이버를 다운로드 하려면](../sql-connection-libraries.md#anchor-20-drivers-relational-access)
 
## <a name="using-the-jdbc-driver-with-maven-central"></a>Maven 중부와 JDBC 드라이버를 사용 하 여
JDBC 드라이버는 아래 코드를 사용 하 여 POM.xml 파일에서 종속성으로 추가 하 여 Maven 프로젝트에 추가할 수 있습니다.

```xml
<dependency>
    <groupId>com.microsoft.sqlserver</groupId>
    <artifactId>mssql-jdbc</artifactId>
    <version>6.2.1.jre8</version>
</dependency>
```

## <a name="unsupported-drivers"></a>지원되지 않는 드라이버  
지원 되지 않는 드라이버 버전은 다운로드할 수 없습니다. Java 연결 지원을 지속적으로 개선하고 있습니다. 따라서 최신 버전의 JDBC 드라이버와 협력 하는 것이 좋습니다.
