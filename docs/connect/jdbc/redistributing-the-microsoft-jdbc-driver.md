# <a name="redistributing-the-microsoft-jdbc-driver"></a>Microsoft JDBC Driver 재배포
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

    
## <a name="register-to-redistribute-the-microsoft-jdbc-40-driver"></a>Microsoft JDBC 4.0 Driver를 재배포하도록 등록  
 4.0 버전의 드라이버에 대한 사용권 계약을 검토하려면 아래 링크를 클릭합니다.  원할 경우 기록을 위해 사용권 계약의 사본을 인쇄하여 보관할 수 있습니다.  
  
 사용권 계약에 동의하면 등록 페이지와 다운로드 페이지로 차례로 이동됩니다.  
  
 [Microsoft JDBC Driver 4.0 for SQL Server에 대 한 재배포 라이선스](https://msdn.microsoft.com/sqlserver/jj589698)  
  
 참고: Microsoft는 보안 인증 및 등록을 위해 Windows Live ID를 사용합니다. Windows Live ID를 통해 SQL Server JDBC Driver 재배포를 위한 재배포 권한을 등록하면 보안 프로필에 포함될 정보를 제공하도록 요구됩니다.  
  
  [Windows Live ID](https://signup.live.com/)가 아직 없으면 받을 수 있습니다.  
  

## <a name="redistribute-the-microsoft-jdbc-41-42-and-60-driver"></a>4.1, 4.2 및 6.0 Microsoft JDBC Driver를 재배포
4.1, 4.2, 6.0 및 6.2 버전의 드라이버에 대 한 사용권 계약에서 "배포 가능 코드" 절을 검토 하십시오.
