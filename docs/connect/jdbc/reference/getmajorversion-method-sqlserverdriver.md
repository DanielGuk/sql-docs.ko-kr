# <a name="getmajorversion-method-sqlserverdriver"></a>getMajorVersion 메서드(SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  주 버전 번호를 반환 합니다.는 [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getMajorVersion()  
```  
  
## <a name="return-value"></a>반환 값  
 **int** JDBC 드라이버의 주 버전 번호를 포함 하 합니다.  
  
## <a name="remarks"></a>주의  
 이 getMajorVersion 메서드는 java.sql.Driver 인터페이스의 getMajorVersion 메서드에 의해 지정 됩니다.
