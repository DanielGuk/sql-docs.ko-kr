# <a name="getsavepointid-method-sqlserversavepoint"></a>getSavepointId 메서드(SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  저장점의 ID를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getSavepointId()  
```  
  
## <a name="return-value"></a>반환 값  
 **int** 값입니다.
