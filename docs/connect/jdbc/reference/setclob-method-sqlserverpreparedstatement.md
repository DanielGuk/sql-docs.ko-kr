# <a name="setclob-method-sqlserverpreparedstatement"></a>setClob 메서드(SQLServerPreparedStatement)

  Java.sql.Clob 개체에 지정된 된 매개 변수를 설정합니다.

|이름|Description|  
|----------|-----------------|  
|[setClob 메서드 &#40; int, java.io.Reader &#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|Java.sql.Clob 개체에 지정된 된 매개 변수를 설정합니다.|
