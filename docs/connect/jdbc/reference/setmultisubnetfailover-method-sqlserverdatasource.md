# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>setMultiSubnetFailover 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  값을 설정 하는 **multiSubnetFailover** 연결 속성입니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>매개 변수  
 *multiSubnetFailover*  
  
 새 값은 **multiSubnetFailover** 연결 속성입니다.  
  
## <a name="remarks"></a>주의  
 에 대 한 자세한 내용은 **multiSubnetFailover** 연결 속성 참조 [연결 속성을 설정할](../../../connect/jdbc/setting-the-connection-properties.md)합니다.
