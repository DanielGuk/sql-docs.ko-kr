# <a name="setlogwriter-method-sqlserverdatasource"></a>setLogWriter 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 메서드는 내부 전용입니다. 로깅에 대 한 자세한 내용은 참조 [드라이버 작업 추적](../../../connect/jdbc/tracing-driver-operation.md)합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public void setLogWriter(java.io.PrintWriter out)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *아웃*  
  
 PrintWriter 개체입니다.  
  
## <a name="remarks"></a>주의  
 이 setLogWriter 메서드는 javax.sql.DataSource 인터페이스의 setLogWriter 메서드에 의해 지정 됩니다.
