# <a name="setshort-method-sqlservercallablestatement"></a>setShort 메서드(SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 된 매개 변수 설정 하는 지정 된 **짧은** 값입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public void setShort(java.lang.String sCol,  
                     short s)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *sCol*  
  
 A **문자열** 매개 변수 이름이 들어 있는입니다.  
  
 *s*  
  
 A **짧은** 값입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 setShort 메서드는 java.sql.CallableStatement 인터페이스의 setShort 메서드에 의해 지정 됩니다.
