# <a name="previous-method-sqlserverresultset"></a>이전 메서드 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 이전 행으로 커서 이동 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean previous()  
```  
  
## <a name="return-value"></a>반환 값  
 **true 이면** 새 현재 행이 유효 합니다. **false** 처리할 행이 더 이상 없는 경우.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 이전 메서드는 java.sql.ResultSet 인터페이스의 이전 메서드에 의해 지정 됩니다.
