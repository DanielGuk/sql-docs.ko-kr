# <a name="gettypemap-method-sqlserverconnection"></a>getTypeMap 메서드(SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 연결 되는 맵 개체를 검색 [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 개체입니다.  
  
> [!NOTE]  
>  이 메서드는 현재 지원 되지 않습니다는 [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)]합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.util.Map getTypeMap()  
```  
  
## <a name="return-value"></a>반환 값  
 지도 개체입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 getTypeMap 메서드는 java.sql.Connection 인터페이스의 getTypeMap 메서드에 의해 지정 됩니다.
