# <a name="executeupdate-method-sqlserverpreparedstatement"></a>executeUpdate 메서드 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 SQL 문을 실행 [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) 개체는 SQL INSERT, UPDATE, MERGE 또는 DELETE 문과 또는 DDL 문 같이 아무 것도 반환 하는 SQL 문을 이어야 합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|Description|  
|----------|-----------------|  
|[executeUpdate)](../../../connect/jdbc/reference/executeupdate-method.md)|이 SQL 문을 실행 [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) 개체는 SQL INSERT, UPDATE, MERGE 또는 DELETE 문과 또는 DDL 문 같이 아무 것도 반환 하는 SQL 문을 이어야 합니다.|  
|[executeUpdate (java.lang.String)](../../../connect/jdbc/reference/executeupdate-method-java-lang-string.md)|INSERT, UPDATE, MERGE 또는 DELETE 문과 같은 지정된 SQL 문이나 SQL DDL 문 같이 아무 것도 반환하지 않는 SQL 문을 실행합니다.|
