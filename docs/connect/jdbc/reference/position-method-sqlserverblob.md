# <a name="position-method-sqlserverblob"></a>position 메서드(SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 패턴과 시작 인덱스에 따라 BLOB에서 지정된 패턴이 있는 위치를 반환합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|Description|  
|----------|-----------------|  
|[position (java.sql.Blob, long)](../../../connect/jdbc/reference/position-method-java-sql-blob-long.md)|지정된 패턴과 시작 인덱스에 따라 BLOB에서 지정된 패턴이 있는 위치를 반환합니다.|  
|[위치 (바이트&#91;&#93;, long)](../../../connect/jdbc/reference/position-method-byte-long.md)|에 따라 BLOB에서 지정된 된 패턴의 위치를 반환 합니다.는 주어진 **바이트** 배열 패턴과 시작 인덱스입니다.|
