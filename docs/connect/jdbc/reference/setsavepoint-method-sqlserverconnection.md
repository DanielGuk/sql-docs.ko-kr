# <a name="setsavepoint-method-sqlserverconnection"></a>setSavepoint 메서드(SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  현재 트랜잭션에 명명되지 않은 저장점을 만들고 이 저장점을 나타내는 새 [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) 개체를 반환합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|설명|  
|----------|-----------------|  
|[setSavepoint ()](../../../connect/jdbc/reference/setsavepoint-method.md)|현재 트랜잭션에 명명되지 않은 저장점을 만들고 이 저장점을 나타내는 새 [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) 개체를 반환합니다.|  
|[setSavepoint(java.lang.String)](../../../connect/jdbc/reference/setsavepoint-method-java-lang-string.md)|현재 트랜잭션에 지정된 이름의 저장점을 만들고 이 저장점을 나타내는 새 [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) 개체를 반환합니다.|
