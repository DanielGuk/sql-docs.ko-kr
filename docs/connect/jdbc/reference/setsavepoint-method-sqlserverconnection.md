# <a name="setsavepoint-method-sqlserverconnection"></a>setSavepoint 메서드 (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  현재 트랜잭션에 명명 되지 않은 저장 점을 만들고 새 반환 [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) 점을 나타내는 개체입니다.

|----------|-----------------|  
|[setSavepoint)](../../../connect/jdbc/reference/setsavepoint-method.md)|현재 트랜잭션에 명명 되지 않은 저장 점을 만들고 새 반환 [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) 점을 나타내는 개체입니다.|  
|[setSavepoint (java.lang.String)](../../../connect/jdbc/reference/setsavepoint-method-java-lang-string.md)|현재 트랜잭션에 지정 된 이름의 저장 점을 만들고 새 반환 [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) 점을 나타내는 개체입니다.|
