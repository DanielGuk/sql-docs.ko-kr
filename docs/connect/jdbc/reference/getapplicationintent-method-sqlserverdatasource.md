# <a name="getapplicationintent-method-sqlserverdatasource"></a>getApplicationIntent 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  값을 반환 된 **applicationIntent** 연결 속성입니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## <a name="return-value"></a>반환 값  
 값을 반환 된 **applicationIntent** 연결 속성입니다.  
  
## <a name="remarks"></a>주의  
 에 대 한 자세한 내용은 **applicationIntent** 연결 속성 참조 [연결 속성을 설정할](../../../connect/jdbc/setting-the-connection-properties.md)합니다.
