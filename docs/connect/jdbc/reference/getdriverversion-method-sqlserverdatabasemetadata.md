# <a name="getdriverversion-method-sqlserverdatabasemetadata"></a>getDriverVersion 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 JDBC 드라이버의 버전 번호를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.lang.String getDriverVersion()  
```  
  
## <a name="return-value"></a>반환 값  
 A **문자열** JDBC 드라이버 버전을 포함 하 합니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 getDriverVersion 메서드는 java.sql.DatabaseMetaData 인터페이스의 getDriverVersion 메서드에 의해 지정 됩니다.
