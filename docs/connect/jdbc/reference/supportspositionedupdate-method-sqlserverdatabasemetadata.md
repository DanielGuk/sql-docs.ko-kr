# <a name="supportspositionedupdate-method-sqlserverdatabasemetadata"></a>supportsPositionedUpdate 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 데이터베이스에서 위치 지정된 UPDATE 문을 지원하는지 여부를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean supportsPositionedUpdate()  
```  
  
## <a name="return-value"></a>반환 값  
 **true 이면** 지원 되는 경우. 그렇지 않으면 **false**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 supportsPositionedUpdate 메서드는 java.sql.DatabaseMetaData 인터페이스의 supportsPositionedUpdate 메서드에 의해 지정 됩니다.
