# <a name="createblob-method-sqlserverconnection"></a>createBlob 메서드(SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  데이터가 없는 Blob 개체를 만듭니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.sql.Blob createBlob()  
```  
  
## <a name="return-value"></a>반환 값  
 Blob 개체입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 createBlob 메서드는 java.sql.Connection 인터페이스의 createBlob 메서드에 의해 지정 됩니다.  
  
 이 메서드 대신 사용할 [SQLServerBlob 생성자 &#40; SQLServerConnection, 바이트 &#41; ](../../../connect/jdbc/reference/sqlserverblob-constructor-sqlserverconnection-byte.md).
