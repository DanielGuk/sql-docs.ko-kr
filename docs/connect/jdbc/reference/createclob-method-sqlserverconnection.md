# <a name="createclob-method-sqlserverconnection"></a>createClob 메서드(SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  데이터가 없는 Clob 개체를 만듭니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.sql.Clob createClob()  
```  
  
## <a name="return-value"></a>반환 값  
 Clob 개체입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 createClob 메서드는 java.sql.Connection 인터페이스의 createClob 메서드에 의해 지정 됩니다.  
  
 이 메서드 대신 사용할 [SQLServerClob 생성자 &#40; SQLServerConnection, java.lang.String &#41; ](../../../connect/jdbc/reference/sqlserverclob-constructor-sqlserverconnection-java-lang-string.md).
