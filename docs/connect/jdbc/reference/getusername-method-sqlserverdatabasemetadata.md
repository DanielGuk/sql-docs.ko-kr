# <a name="getusername-method-sqlserverdatabasemetadata"></a>getUserName 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 데이터베이스에 알려진 사용자 이름을 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public java.lang.String getUserName()  
```  
  
## <a name="return-value"></a>반환 값  
 A **문자열** 사용자 이름이 들어 있는입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 getUserName 메서드는 java.sql.DatabaseMetaData 인터페이스의 getUserName 메서드에 의해 지정 됩니다.
