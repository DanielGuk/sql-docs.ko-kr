# <a name="first-method-sqlserverresultset"></a>첫 번째 메서드 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 첫 번째 행으로 커서 이동 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean first()  
```  
  
## <a name="return-value"></a>반환 값  
 **true 이면** 커서가 첫 번째 행으로 이동 합니다. 그렇지 않으면 **false**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 첫 번째 메서드는 java.sql.ResultSet 인터페이스의 첫 번째 메서드에 의해 지정 됩니다.
