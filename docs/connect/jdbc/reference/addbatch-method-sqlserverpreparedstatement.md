# <a name="addbatch-method-sqlserverpreparedstatement"></a>addBatch 메서드 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 매개 변수 집합의 명령 일괄 처리에 추가 [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) 개체입니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|Description|  
|----------|-----------------|  
|[addBatch)](../../../connect/jdbc/reference/addbatch-method.md)|이 명령 일괄 처리에 매개 변수 집합 추가 [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) 개체입니다.|  
|[addBatch (java.lang.String)](../../../connect/jdbc/reference/addbatch-method-java-lang-string.md)|이 대 한 명령의 현재 목록에 지정된 된 SQL 명령을 추가 [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) 개체입니다.|
