# <a name="getmultisubnetfailover-method-sqlserverdatasource"></a>getMultiSubnetFailover 메서드(SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  값을 반환 된 **multiSubnetFailover** 연결 속성입니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
public boolean getMultiSubnetFailover();  
```  
  
## <a name="return-value"></a>반환 값  
 연결 속성의 현재 설정에 따라 true 또는 false를 반환합니다.

 에 대 한 자세한 내용은 **multiSubnetFailover** 연결 속성 참조 [연결 속성을 설정할](../../../connect/jdbc/setting-the-connection-properties.md)합니다.
