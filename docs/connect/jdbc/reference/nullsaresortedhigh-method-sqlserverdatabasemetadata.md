# <a name="nullsaresortedhigh-method-sqlserverdatabasemetadata"></a>nullsAreSortedHigh 메서드(SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  NULL 값이 상위에 정렬되는지 여부를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean nullsAreSortedHigh()  
```  
  
## <a name="return-value"></a>반환 값  
 **true 이면** 경우 값이 상위에 정렬 됩니다. 그렇지 않으면 **false**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 nullsAreSortedHigh 메서드는 java.sql.DatabaseMetaData 인터페이스의 nullsAreSortedHigh 메서드에 의해 지정 됩니다.
