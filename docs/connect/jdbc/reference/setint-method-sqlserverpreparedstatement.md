# <a name="setint-method-sqlserverpreparedstatement"></a>setInt 메서드 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 된 매개 변수 설정 하는 지정 된 **int** 값입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public final void setInt(int n,  
                         int value)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *n*  
  
 **int** 매개 변수 수를 나타내는입니다.  
  
 *value*  
  
 **int** 값입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 setInt 메서드는 java.sql.PreparedStatement 인터페이스의 setInt 메서드에 의해 지정 됩니다.
