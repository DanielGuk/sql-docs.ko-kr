# <a name="updatebigdecimal-method-sqlserverresultset"></a>updateBigDecimal 메서드(SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  BigDecimal 개체와 지정된 된 열을 업데이트합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|속성|설명|  
|----------|-----------------|  
|[updateBigDecimal(int, java.math.BigDecimal)](../../../connect/jdbc/reference/updatebigdecimal-method-int-java-math-bigdecimal.md)|열 인덱스가 지정된 경우 지정된 열을 BigDecimal 개체로 업데이트합니다.|  
|[updateBigDecimal(java.lang.String, java.math.BigDecimal)](../../../connect/jdbc/reference/updatebigdecimal-method-java-lang-string-java-math-bigdecimal.md)|열 이름이 지정된 경우 지정된 열을 BigDecimal 개체로 업데이트합니다.|
