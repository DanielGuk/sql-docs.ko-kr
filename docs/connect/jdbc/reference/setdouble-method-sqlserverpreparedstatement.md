# <a name="setdouble-method-sqlserverpreparedstatement"></a>setDouble 메서드 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정 된 매개 변수를 설정 된 주어진 **double** 값입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public final void setDouble(int n,  
                            double x)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *n*  
  
 **int** 매개 변수 수를 나타내는입니다.  
  
 *x*  
  
 A **double** 값입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 setDouble 메서드는 java.sql.PreparedStatement 인터페이스의 setDouble 메서드에 의해 지정 됩니다.
