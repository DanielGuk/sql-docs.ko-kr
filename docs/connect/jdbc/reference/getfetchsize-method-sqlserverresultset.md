# <a name="getfetchsize-method-sqlserverresultset"></a>getFetchSize 메서드 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  이 대 한 인출 크기를 검색 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public int getFetchSize()  
```  
  
## <a name="return-value"></a>반환 값  
 **int** 현재 인출 크기를 나타내는입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>주의  
 이 getFetchSize 메서드는 java.sql.ResultSet 인터페이스의 getFetchSize 메서드가 지정 됩니다.
