# <a name="sqlserversavepoint-constructor-sqlserverconnection-javalangstringname"></a>SQLServerSavepoint 생성자(SQLServerConnection, java.lang.StringName)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  새 인스턴스를 초기화는 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) 클래스 지정 된 연결 및 이름에 기반 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public SQLServerSavepoint(SQLServerConnection con,  
                          java.lang.String sName)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *con*  
  
 A [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 개체입니다.  
  
 *sName*  
  
 A **문자열** 저장 점의 이름이 들어 있는입니다.
