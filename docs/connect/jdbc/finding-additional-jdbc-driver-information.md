# <a name="finding-additional-jdbc-driver-information"></a>추가 JDBC 드라이버 정보 찾기
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  에 대 한 자세한 내용은 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] 및 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 개발 일반적으로 다음 온라인 리소스를 참조 하십시오.  
  
## <a name="remarks"></a>주의  
  
|리소스|Description|  
|--------------|-----------------|  
|[SQL Server GitHub 리포지토리에 대 한 JDBC 드라이버](https://github.com/microsoft/mssql-jdbc)|이 저장소는 SQL Server 용 JDBC 드라이버에 대 한 소스 코드를 포함합니다. 파일 문제를이 사이트를 사용 하 여 있으며 직접 드라이버를 합니다.|
|[데이터 액세스 및 저장소 개발자 센터](http://go.microsoft.com/fwlink?linkid=4173)|이 사이트는 모든 데이터 액세스 기술에 대 한 설명서, 기술 문서, 예제 코드 및 기타 리소스를 제공 [!INCLUDE[msCoName](../../includes/msconame_md.md)]합니다.|  
|[SQL Server 데이터 액세스 포럼](http://go.microsoft.com/fwlink/?LinkId=70651)|이 사이트 역할에 대 한 데이터 액세스에 대 한 커뮤니티 포럼을 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 를 사용 하 여 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] Native Client, OLE DB, ODBC, ADO, MDAC, JDBC 또는 SOAP/HTTP입니다.|  
|[JDBC 블로그](http://go.microsoft.com/fwlink/?LinkId=124746)|이 블로그에서에 대 한 정보를 제공 하는 데 사용 되는 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] 및 JDBC 드라이버 제품 팀의 팀원과 직접 대화할 수 있습니다.|  
|[Microsoft Connect](http://go.microsoft.com/fwlink/?LinkID=116905)|이 사이트를 통해 다른 고객이 제출한 드라이버 문제를 검색하고 본인의 의견도 제출할 수 있습니다.|
