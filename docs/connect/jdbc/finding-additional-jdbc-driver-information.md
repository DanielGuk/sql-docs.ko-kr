# <a name="finding-additional-jdbc-driver-information"></a>추가 JDBC 드라이버 정보 찾기
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  전반적인 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] 및 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 개발에 대한 자세한 내용은 다음 온라인 리소스를 참조하십시오.  
  
## <a name="remarks"></a>Remarks  
  
|리소스|설명|  
|--------------|-----------------|  
|[SQL Server GitHub 리포지토리에 대 한 JDBC 드라이버](https://github.com/microsoft/mssql-jdbc)|이 리포지토리는 SQL Server 용 JDBC 드라이버에 대 한 소스 코드를 포함합니다. 파일 문제를이 사이트를 사용 하 고 드라이버에 직접적인 영향을 합니다.|
|[데이터 액세스 및 저장소 개발자 센터](http://go.microsoft.com/fwlink?linkid=4173)|이 사이트에서는 [!INCLUDE[msCoName](../../includes/msconame_md.md)]의 모든 데이터 액세스 기술에 대한 설명서, 기술 문서, 예제 코드 및 기타 리소스를 제공합니다.|  
|[SQL Server 데이터 액세스 포럼](http://go.microsoft.com/fwlink/?LinkId=70651)|이 사이트는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client, OLE DB, ODBC, ADO, MDAC, JDBC 또는 SOAP/HTTP를 사용한 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 데이터 액세스에 대한 커뮤니티 포럼 역할을 합니다.|  
|[JDBC 블로그](http://go.microsoft.com/fwlink/?LinkId=124746)|이 블로그에서는 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)]에 관한 정보가 제공되며 JDBC 드라이버 제품 팀의 팀원과 직접 대화할 수 있습니다.|  
|[Microsoft Connect](http://go.microsoft.com/fwlink/?LinkID=116905)|이 사이트를 통해 다른 고객이 제출한 드라이버 문제를 검색하고 본인의 의견도 제출할 수 있습니다.|
