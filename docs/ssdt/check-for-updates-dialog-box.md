# <a name="check-for-updates-dialog-box"></a>업데이트 확인 대화 상자
SQL Server Data Tools의 2012년 11월 릴리스부터는 **SQL** 메뉴 아래에 **업데이트 확인** 메뉴 옵션이 표시됩니다.  
  
**업데이트 확인** 대화 상자에는 컴퓨터에 설치되어 있는 SQL Server Data Tools 버전과 릴리스 버전이 표시됩니다. 릴리스 버전이 설치된 버전보다 최신 버전인 경우 **업데이트**가 표시됩니다. **업데이트** 단추를 클릭하여 사용 가능한 업데이트에 대한 설명을 참조하세요. 업데이트를 선택하면 최신 SQL Server Data Tools 버전을 컴퓨터에 설치합니다.  
  
**업데이트 확인** 대화 상자에는 **SQL Server Data Tools에 대한 업데이트 자동으로 확인** 확인란도 있습니다. 확인란을 선택하면 Visual Studio를 시작할 때 사용 가능한 업데이트에 대한 설명이 SQL Server Data Tools에 표시됩니다. 이러한 업데이트를 설치하도록 선택할 수 있습니다.
