# <a name="lesson-2-5---summary---configuring-permissions-on-database-objects"></a>2-5단원 - 요약 - 데이터베이스 개체에 대한 액세스 권한 부여
[!INCLUDE[tsql-appliesto-ss2008-all-md](../includes/tsql-appliesto-ss2008-all-md.md)] 로그인은 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]에 연결할 수 있는 권한을 사용자에게 제공합니다. 사용자는 특정 데이터베이스에 액세스할 수 있는 로그인입니다. GRANT 문을 사용하여 데이터 읽기, 액세스 및 변경을 수행할 수 있는 권한을 사용자에게 제공할 수 있습니다.  
  
뷰는 단일 SELECT 문이며 사용자에게 테이블처럼 표시됩니다. 저장 프로시저는 일괄 처리로 실행되는 하나 이상의 [!INCLUDE[tsql](../includes/tsql-md.md)] 문입니다.
