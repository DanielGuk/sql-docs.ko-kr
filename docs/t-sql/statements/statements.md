# <a name="transact-sql-statements"></a>Transact-SQL 문
[!INCLUDE[tsql-appliesto-ss2008-asdb-asdw-pdw-_md](../../includes/tsql-appliesto-ss2008-asdb-asdw-pdw-md.md)]

이 참조 항목에는 TRANSACT-SQL (T-SQL)와 함께 사용할 명령문 범주 요약 되어 있습니다. 모든 왼쪽 탐색 창에서 나열 된 문을 찾을 수 있습니다.

## <a name="backup-and-restore"></a>Backup 및 Restore 메서드
Backup 및 restore 문이 백업을 만들고 백업에서 복원 하는 방법을 제공 합니다.  자세한 내용은 참조는 [백업 및 복원 개요](../../relational-databases/backup-restore/back-up-and-restore-of-sql-server-databases.md)합니다.

## <a name="data-definition-language"></a>데이터 정의 언어
데이터 정의 언어 (DDL) 문을 데이터 구조를 정의 합니다. 이러한 문을 사용 하 여 생성, 변경 또는 데이터베이스의 데이터 구조를 삭제 합니다.
- ALTER
- 데이터 정렬
- CREATE
- DROP
- 트리거를 사용 하지 않도록 설정
- ENABLE TRIGGER
- 이름 바꾸기
- UPDATE STATISTICS

## <a name="data-manipulation-language"></a>데이터 조작 언어
데이터 조작 언어 (DML) 데이터베이스에 저장 된 정보를 영향을 줍니다. 이러한 문을 사용 하 여 삽입, 업데이트 및 데이터베이스의 행을 변경 합니다.

- BULK INSERT
- DELETE
- INSERT
- MERGE
- TRUNCATE TABLE

## <a name="permissions-statements"></a>사용 권한 문
사용 권한 문을 사용자와 로그인을 데이터에 액세스 하 고 작업을 수행할 수를 결정 합니다. 인증 및 액세스 하는 방법에 대 한 자세한 내용은 참조는 [보안 센터](../../relational-databases/security/security-center-for-sql-server-database-engine-and-azure-sql-database.md)합니다.

## <a name="service-broker-statements"></a>Service Broker 문
Service Broker는 메시징 및 큐 응용 프로그램에 대 한 기본 지원을 제공 하는 기능입니다. 자세한 내용은 참조 [Service Broker](../../relational-databases/service-broker/event-notifications.md)합니다.

## <a name="session-settings"></a>세션 설정
SET 문을 현재 세션 핸들에서 시간 설정을 실행 하는 방법을 결정 합니다. 에 대 한 개요 [SET 문을](set-statements-transact-sql.md)합니다.
