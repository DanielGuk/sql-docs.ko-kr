# <a name="continue-transact-sql"></a>CONTINUE(Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-asdb-xxxx-xxx_md](../../includes/tsql-appliesto-ss2008-asdb-xxxx-xxx-md.md)]

  WHILE 루프를 다시 시작합니다. CONTINUE 키워드 다음에 나오는 문은 모두 무시됩니다. CONTINUE는 항상은 아니지만 IF 테스트에 의해 실행되는 경우가 많습니다. 자세한 내용은 참조 [동안 &#40; Transact SQL &#41; ](../../t-sql/language-elements/while-transact-sql.md) 및 [흐름 제어 언어 &#40; Transact SQL &#41; ](~/t-sql/language-elements/control-of-flow.md).
