# <a name="odbc-test"></a>ODBC 테스트
Microsoft® ODBC 테스트는 ODBC 드라이버 및 ODBC 드라이버 관리자를 테스트 하는 데 사용할 수 있는 ODBC 사용 응용 프로그램입니다. ANSI 및 유니코드 지원 버전 ODBC 테스트의 ODBC 3.51에 포함 되어 있습니다. 해당 파일은 다음과 같습니다.  
  
-   Odbcte32.exe 및 Gtrtst32.dll ANSI 버전에 대 한 합니다.  
  
-   Odbct32w.exe 및 Gtrts32w.dll 유니코드 버전에 대 한 합니다.  
  
 ODBC 테스트를 사용 하려면 ODBC API는 C 언어와 SQL 이해 해야 합니다. ODBC API에 대 한 자세한 내용은 참조는 [ODBC Programmer's Reference](../odbc/reference/odbc-programmer-s-reference.md)합니다.  
  
 이제 프로그램을 ODBC 테스트 이전의 설명서의이 섹션에 포함 된 도움말 항목이 포함 되어 있습니다. Odbcte32.exe 열거나 Odbct32w.exe, 열기는 **도움말** 메뉴를 차례로 클릭 **도움말 항목**합니다.  
  
 별도 파일에도 64 비트 Microsoft Windows 운영 체제를 위한 응용 프로그램을 64 비트 버전의 32 비트 버전으로 동일한 이름을 가졌는지 note 합니다. 즉, 64 비트 버전의 ODBC 테스트의 유니코드 버전에 대 한 이름은 odbct32w.exe입니다.
