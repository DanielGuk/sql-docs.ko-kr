# <a name="choosing-a-level-of-interoperability"></a>상호 운용성 수준을 선택
다음 단계에서는 어떤 수준의 상호 운용성이 필요한를 결정 하는 것으로 가정 하 고 응용 프로그램에서 ODBC를 사용 합니다. 기본 수준의 상호 운용성 응용 프로그램 종류의 함수는 일반적으로: 사용자 지정 응용 프로그램 경향이 하지 상호 운용 가능 수직 응용 프로그램을 제한 된 수의 Dbms, 간의 상호 운용 가능한 경향이 및 상호 운용할 수 있도록 하는 경향이 일반 응용 프로그램 모든 Dbms 중.
