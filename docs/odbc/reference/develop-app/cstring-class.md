# <a name="cstring-class"></a>CString 클래스
때문에 개체에는 **CString** Microsoft® C++®에서 클래스 서명 되 고 ODBC 함수에 문자열 인수는 서명 되지 전달 하는 응용 프로그램 **CString** 개체 없이 ODBC 함수를 이러한 캐스팅 컴파일러 경고를 받게 됩니다.
