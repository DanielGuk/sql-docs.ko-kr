# <a name="enabling-visual-studio-analyzer"></a>Visual Studio Analyzer를 사용 하도록 설정
> [!IMPORTANT]  
>  Visual Studio Analyzer에 대 한 지원 (Visual Studio Analyzer 이전 버전의 Visual Studio에만 포함 되었습니다.) Windows 8 부터는 제거 되었습니다. 다른 방법은 문제 해결 메커니즘 BID 추적을 사용 합니다.  
  
 Microsoft Visual Studio Analyzer는 모든 계층 및 시스템에서 자세히 살펴봅니다 응용 프로그램의 성능 제공 하도록 설계 된 응용 프로그램 분석 도구입니다. 구성 요소 간의 상호 작용에 중점을 둡니다. Visual Studio Analyzer는 시작 하 고 ODBC 관리자 추적 탭에서 중지할 수 있습니다.  
  
 ODBC를 생성 하는 이벤트를 보려면 다음을 수행 합니다.  
  
1.  ODBC 데이터 원본 관리자에 액세스 합니다.  
  
2.  클릭는 **추적** 탭 합니다.  
  
3.  클릭 **Visual Studio Analyzer 추적 시작**합니다.  
  
4.  **확인**을 클릭합니다.  
  
 Visual Studio Analyzer 이벤트 생성 될 때까지 계속 **중지 Visual Studio Analyzer 추적** 을 선택 합니다.  
  
 이 섹션에는 다음 항목 포함 되어 있습니다.  
  
-   [ODBC 드라이버 관리자에 의해 생성된 이벤트](../../../odbc/reference/develop-app/events-generated-by-the-odbc-driver-manager.md)
