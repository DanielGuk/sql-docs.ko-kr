SQL 문의 일괄 처리에 대 한 지원, 광범위 한 하지 않으므로 상호 운용 가능한 응용 프로그램 조건에 따라 되지 않거나 전혀을 사용 해야 합니다. 자세한 내용은 참조 [일괄 처리 실행](../../../odbc/reference/develop-app/executing-batches.md)합니다.
