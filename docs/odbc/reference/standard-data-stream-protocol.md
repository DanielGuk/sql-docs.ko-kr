# <a name="standard-data-stream-protocol"></a>표준 데이터 스트림 프로토콜
표준 데이터 스트림 프로토콜을 수 있는 방법은 유형이 다른 Dbms의 데이터에 액세스 합니다. 실제로 표준 데이터 스트림 프로토콜은 이미: ANSI/ISO 원격 데이터베이스 액세스 RDA () 표준 ([ISO/IEC 9579:2000](http://www.iso.org/iso/catalogue_detail.htm?csnumber=30615))를 IBM DRDA 분산 관계형 데이터베이스 아키텍처 () 및 ([ http://www.netstoreusa.com/cbbooks/007/0070351384.shtml ](http://www.netstoreusa.com/cbbooks/007/0070351384.shtml)). 두 시스템 약속을 표시 하지만 모두 지금 구현 광범위 하 게 됩니다.
