# <a name="connection-transitions"></a>연결 전환
ODBC 연결 상태를 갖습니다.  
  
|State|Description|  
|-----------|-----------------|  
|C0|연결 할당 되지 않은 할당 되지 않은 환경|  
|C1|연결 할당 되지 않은 할당 된 환경|  
|C2|환경 연결에 할당 된 할당|  
|C3|연결 함수에 필요한 데이터|  
|C4|연결 된 연결|  
|C5|문에 할당 된 연결을 연결 합니다.|  
|C 6|연결 된 연결, 트랜잭션 진행 중에서입니다. 연결에 할당 되지 문으로 C6 상태인 것으로 연결에 대 한 것 같습니다. 예를 들어 연결 수동 커밋 모드에 있고 C4 상태에 있습니다. 문을 할당, (트랜잭션 시작)를 실행 및 다음 해제 하는 경우 트랜잭션은 활성 상태로 남아 있지만 연결에 문이 없습니다.|
