# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations 및 커서 라이브러리
> [!IMPORTANT]  
>  이 기능은 나중 버전의 Windows에서 제거 됩니다. 새 개발 작업에서는이 기능을 사용 하지 마십시오 하 고 현재이 기능을 사용 하는 응용 프로그램은 수정 하세요. 드라이버의 커서 기능을 사용 하는 것이 좋습니다.  
  
 커서 라이브러리가 호출을 지원 하지 않을 **SQLBulkOperations**합니다. 에 대 한 일반적인 내용은 **SQLBulkOperations**, 참조 [SQLBulkOperations 함수](../../../odbc/reference/syntax/sqlbulkoperations-function.md)합니다.
