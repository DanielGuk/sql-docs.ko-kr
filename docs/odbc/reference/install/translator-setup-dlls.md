# <a name="translator-setup-dlls"></a>변환기 설치 Dll
> [!NOTE]  
>  Windows XP 및 Windows Server 2003부터 ODBC Windows 운영 체제에 포함 됩니다. 이전 버전의 Windows에서 ODBC만 명시적으로 설치 해야 합니다.  
  
 DLL에 들어 변환기 설치는 **ConfigTranslator** 는 변환기에 대 한 기본 옵션을 반환 하는 함수입니다. 필요한 경우 사용자에 게이 정보 메시지 표시 합니다. 에 대 한 전체 설명은이 함수를 참조 하세요. [설치 DLL API 참조](../../../odbc/reference/syntax/setup-dll-api-reference.md)합니다.  
  
 변환기 설치 DLL 변환기 개발자가 작성 됩니다. 변환기의 일부가 될 수 있습니다 DLL 또는 별도 DLL입니다.
