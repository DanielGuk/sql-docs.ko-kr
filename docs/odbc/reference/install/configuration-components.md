# <a name="configuration-components"></a>구성 요소
> [!NOTE]  
>  Windows XP 및 Windows Server 2003부터 ODBC Windows 운영 체제에 포함 됩니다. 이전 버전의 Windows에서 ODBC만 명시적으로 설치 해야 합니다.  
  
 데이터 원본 Dll 및 변환기 설치 Dll이 필요에 따라 turn 호출 드라이버에서를 설치 하는 DLL의 경우 설치 관리자가 구성 됩니다. DLL 중 하나에서 직접 호출 또는 로드 되 고 라는 다른 프로그램에 의해 호출 설치 관리자는 *관리 프로그램*합니다. 다음 그림은 구성 요소 간의 관계를 보여 줍니다.  
  
 ![구성 요소 간의 관계](../../../odbc/reference/install/media/pr30.gif "pr30")  
  
 이러한 구성 요소에 대 한 자세한 내용은이 섹션의 끝에 다음 항목을 참조 합니다.
