# <a name="default-subkey"></a>기본 하위 키
레지스트리 기본 하위 키를 기본 데이터 원본을 지정할 수 있습니다. 이 하위 키는 특별 한 경우 데이터 원본 사양 하위 키의 있고 다른 데이터 원본 사양 하위 키와 같은 값을 보유 합니다. 유일한 차이점은 ODBC 데이터 원본 하위 키 아래에 있는 값으로 나열 되지 않아서 합니다.
