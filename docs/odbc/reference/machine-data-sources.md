# <a name="machine-data-sources"></a>데이터 원본
*데이터 소스 컴퓨터* 사용자 정의 이름으로 시스템에 저장 됩니다. 데이터 원본 이름과 연결 된 데이터 원본에 연결 드라이버 관리자와 드라이버 필요성을은 모든 정보입니다. Xbase 데이터 원본에 대 한이 Xbase 드라이버 Xbase 파일과 드라이버를 해당 파일의 예: 단일 사용자 모드 또는 읽기 전용으로 사용 하는 방법을 지시 하는 몇 가지 옵션을 포함 하는 디렉터리의 전체 경로 이름을 수 있습니다. Oracle 데이터 원본에 대 한이 Oracle 드라이버를 Oracle DBMS 있는 서버는 SQL의 이름이 될 수 * SQL를 식별 하는 연결 문자열을 Net\*를 사용 하려면 Net 드라이버 및 서버에서 데이터베이스의 시스템 ID입니다.
