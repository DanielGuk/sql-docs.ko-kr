# <a name="odbc-driver-for-oracle-conformance-levels"></a>Oracle 받는 규칙 수준에 대 한 ODBC 드라이버
> [!IMPORTANT]  
>  이 기능은 나중 버전의 Windows에서 제거 됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 응용 프로그램은 수정하세요. Oracle에서 제공 하는 ODBC 드라이버를 사용 하십시오.  
  
 ODBC 드라이버에 대 한 규칙에 따라 표준의 두 가지 형식을 정의: SQL 문법 규칙 표준 및 표준 API 적합성 합니다. API를 적용 한 드라이버가 지 원하는 함수를 참조 합니다. SQL 규격 드라이버에서 지 원하는 SQL 문법 참조 합니다. 표준 각 규칙은 수준으로 이루어져 있습니다.  
  
 이 섹션에서는 Oracle에 대 한 ODBC 드라이버에 대 한 준수 정보를 제공 합니다.
