# <a name="index-name-limitations"></a>인덱스 이름 제한 사항
Paradox 드라이버를 사용 하는 경우 기본 인덱스는 정의 된 테이블 이름이 같은 있어야 합니다. 다른 고유 하거나 고유 하지 않은 인덱스에는 정의 되어 있는 테이블로 동일한 이름을 가져야 합니다.
