# <a name="jet-40-uses-sql-92-reserved-words-list-when-extendedansisqlset"></a>Jet 4.0 사용 하 여 s Q l 92 예약 된 단어 목록의 ExtendedAnsiSQL_Set
ExtendedAnsiSQL 플래그를 설정 하는 경우 Jet 4.0 SQL 92 예약 된 단어 목록을 사용 합니다. SQL-92 사용 하려고 예약어으로 따옴표로 표시 되지 않은 개체 이름을 구문 오류가 발생 합니다. ExtendedAnsiSQL 플래그를 해제 하 고, 앞으로 개체 이름으로 새 예약어를 사용할 수 있습니다.
