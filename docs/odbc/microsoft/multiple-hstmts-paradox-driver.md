# <a name="multiple-hstmts-paradox-driver"></a>여러 hstmts (Paradox 드라이버)
Paradox ODBC 드라이버 사용 되는 경우 여러 개를 사용 하려는 경우 *hstmt* 테이블에서 쿼리를 실행 하는 테이블에는 고유 인덱스 (기본 키 Paradox) 있어야 합니다.
