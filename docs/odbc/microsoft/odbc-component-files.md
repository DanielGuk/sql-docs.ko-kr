# <a name="odbc-component-files"></a>ODBC 구성 요소 파일
ODBC 데스크톱 데이터베이스 드라이버 4.0의 각 구성 요소에는 ODBC 드라이버 파일이 필요 합니다.  
  
 일부 파일은 둘 이상의 구성 요소가 필요 합니다. 상용 응용 프로그램과 함께 ODBC 데스크톱 데이터베이스 드라이버 중 하나를 재배포 하려는 경우 이러한 파일을 배포 해야 합니다.  
  
 이러한 파일은 Windows 2000 및 Windows NT 3.51 이상 \Windows\System32 디렉터리에서 이상에 대 한 Windows 95, \Windows\System 디렉터리에 설치 됩니다. ODBC 파일이 다른 디렉터리에 이전에 설치 된, 최신 \Windows\System (또는 System32)의 파일 디렉터리를 사용 해야 합니다.
