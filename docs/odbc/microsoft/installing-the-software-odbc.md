# <a name="installing-the-software-odbc"></a>소프트웨어 (ODBC) 설치
> [!IMPORTANT]  
>  이 기능은 나중 버전의 Windows에서 제거 됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 응용 프로그램은 수정하세요. Oracle에서 제공 하는 ODBC 드라이버를 사용 하십시오.  
  
 ODBC Driver for Oracle 데이터 액세스 구성 요소 중 하나입니다. ODBC 데이터 원본 관리자와 같은 다른 ODBC 구성 요소를 포함 하 고 이미 설치 되어 있어야 합니다. 드라이버도 있습니다 "드라이버 및 기타 다운로드 에서" www.microsoft.com에서 Microsoft 기술 지원 서비스 온라인 웹 사이트에 있습니다.  
  
 자체 설명서에 따라 네트워크 소프트웨어를 설치 해야 합니다. Oracle에 대 한 ODBC 드라이버는 네트워크 소프트웨어는 지원으로 특별 한 설치 고려 사항 없이 필요 합니다.  
  
 자체 설명서에 따라 oracle 소프트웨어를 설치 해야 합니다. ODBC Driver for Oracle 드라이버 버전을 지 원하는 특별 한 설치 고려 사항 없이 일반적으로 필요 합니다. 그러나 제품의 호환을 유지 하기 위해 최신 버전의 드라이버를 보관할 수 마지막 Oracle에 대 한 ODBC 드라이버를 설치 합니다. Oracle에는 공용 FTP 사이트를 게시, 특히, 패치 Oracle 서버 제품 및 서버 제품과 함께 제공 되는 클라이언트 구성 요소를 유지 관리 합니다. 이 패치는 여러 Microsoft 제품 및 기술 작동 하는 데 필요 합니다. 이 사이트에 대 한 자세한 내용은 참조 [Oracle 소프트웨어 패치](../../odbc/microsoft/oracle-software-patches.md)합니다.  
  
> [!CAUTION]  
>  MDAC/Windows DAC를 통해 Oracle 소프트웨어를 설치 하면 현재 버전의 MDAC 덮어쓸 수 있습니다. ODBC 구성 요소를 사용 하 여 문제가 발생 하면 다시 MDAC를 설치 합니다.
