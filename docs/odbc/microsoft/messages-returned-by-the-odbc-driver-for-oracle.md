# <a name="messages-returned-by-the-odbc-driver-for-oracle"></a>Oracle에 대 한 ODBC 드라이버에서 반환 되는 메시지
> [!IMPORTANT]  
>  이 기능은 나중 버전의 Windows에서 제거 됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 응용 프로그램은 수정하세요. Oracle에서 제공 하는 ODBC 드라이버를 사용 하십시오.  
  
 Oracle 오류 메시지가 표시 되 면 반송 됩니다 앞에 사용 [Microsoft] [Oracle에 대 한 ODBC 드라이버] 및 [Oracle] 태그; 그렇지 않으면 메시지는 다음 예와 같이 [Oracle] 태그 없이 반환 됩니다.  
  
## <a name="oracle-error-message"></a>Oracle 오류 메시지:  
  
```  
[Microsoft][ODBC driver for Oracle][Oracle]ORA-nnnnn message-text  
```  
  
## <a name="odbc-driver-for-oracle-error-message"></a>Oracle 오류 메시지에 대 한 ODBC 드라이버.  
  
```  
[Microsoft][ODBC driver for Oracle]  
```
