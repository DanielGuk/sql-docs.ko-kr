# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC 드라이버 설정
다운로드 하 고 Visual FoxPro ODBC 드라이버를 설치 하는 방법에 대 한 정보를 참조 하십시오. [Visual FoxPro ODBC 드라이버](http://go.microsoft.com/fwlink/?LinkId=121318) 에서 MSDN 다운로드 합니다.  
  
 Microsoft Visual FoxPro ODBC 드라이버 설치 프로그램을 사용 하 여 다음을 수행 합니다.  
  
-   새 구성 요소를 추가 합니다.  
  
-   구성 요소를 제거 합니다.  
  
-   누락 된 파일 및 설정을 복원 하려면 다시 설치 합니다.  
  
-   이전에 설치 된 모든 구성 요소를 제거 합니다.  
  
 시스템에 드라이버를 설치한 후 설치 프로그램이 설치 된 드라이버 구성 요소를 인식 하 고 드라이버의 구성을 변경할 수 있도록 하는 추가 대화 상자를 표시 합니다.
