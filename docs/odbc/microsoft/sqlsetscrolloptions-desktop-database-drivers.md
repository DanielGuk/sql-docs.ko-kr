정방향 및 정적 커서 SQL_CONCUR_READ_ONLY 지원 됩니다.  
  
 만 키 집합 커서에 대해 지원 됩니다는 *fConcurrency* SQL_CONCUR_LOCK의 인수입니다.  
  
 *fConcurrency* SQL_CONCUR_ROWVER의 인수가 지원 되지 않습니다.
