# <a name="convert-function-limitations"></a>함수 제한 사항으로 변환
변환 오류 결과 NULL로 설정 되 고 영향을 받는 열에 입력 합니다.  
  
 날짜 아니고 타임 스탬프 데이터 형식은 CONVERT 함수에 의해 다른 데이터 형식 (또는 자체)를 변환할 수 있습니다.
