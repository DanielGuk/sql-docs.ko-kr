# <a name="data-truncation-detection-enabled-using-extendedansisql"></a>데이터 잘림 검색 ExtendedAnsiSQL를 사용 하 여 활성화
ExtendedAnsiSQL 플래그가 켜져 하 고 문자 또는 이진 열에는 응용 프로그램 데이터를 삽입 하는 데이터는 잘립니다를 잘림을 검색 됩니다. ExtendedAnsiSQL 플래그를 해제 하 고, 이전 버전의 ODBC 데스크톱 데이터베이스 드라이버에 포함 되어 경고 없이 데이터가 잘립니다.
