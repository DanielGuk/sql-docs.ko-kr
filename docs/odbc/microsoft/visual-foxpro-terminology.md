# <a name="visual-foxpro-terminology"></a>Visual FoxPro 용어
**데이터베이스**  
 Visual FoxPro에 데이터베이스 파일 확장명이.dbc 및 하나 이상 포함 될 수 **테이블**합니다.  
  
 **데이터베이스 테이블**  
 Visual FoxPro에 테이블와 연결 된 데이터베이스입니다. 대비 **자유 테이블**합니다.  
  
 **사용 가능한 테이블**  
 Visual FoxPro 테이블 없는에 데이터베이스와 연결 합니다.  
  
 생성 되는.dbf 파일 FoxPro 버전에서 2.x 테이블인 무료 Visual FoxPro 테이블으로 변환 되 고 Visual FoxPro 데이터베이스에 추가 하지 않는 한 합니다. 대비 **데이터베이스 테이블**합니다.  
  
 **준비할 수 있는 SQL 문**  
 아직 처리 되지 않은 SQL 문에 **SQLPrepare** 함수입니다. Visual FoxPro ODBC 드라이버와 함께이 함수에 대 한 자세한 내용은 참조 [SQLPrepare (Visual FoxPro ODBC 드라이버)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)합니다.  
  
 **table**  
 Visual FoxPro에서 레코드 테이블에 저장 됩니다. 테이블의 각 열은 레코드를 나타내며 테이블의 각 행은 테이블의 열은 레코드의 필드를 나타냅니다. 각 Visual FoxPro 테이블 파일.dbf 확장명 자체에 저장 됩니다. Visual FoxPro 테이블은 데이터베이스와 연결할 수 있습니다.  
  
 FoxPro 버전 2입니다. *x* 테이블은 데이터베이스와 연결 되지 않습니다.
