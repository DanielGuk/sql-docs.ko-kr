# <a name="where-clause-limitations"></a>여기서 절 제한 사항
절 WHERE 절에는 최대 수는 40입니다.  
  
 LONGVARCHAR 및 LONGVARBINARY 열은 최대 255 자 리터럴을 비교할 수 있지만 매개 변수를 사용 하 여 비교할 수 없습니다.
