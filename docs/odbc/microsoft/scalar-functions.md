# <a name="scalar-functions"></a>스칼라 함수
Microsoft ODBC 데스크톱 데이터베이스 드라이버는 다음의 스칼라 함수를 지원합니다.  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 인수 및 반환 값의 스칼라 함수에 대 한 내용은의 부록 F를 참조 하십시오.는 *ODBC 프로그래머 참조 합니다.*
