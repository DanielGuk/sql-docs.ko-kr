# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC 드라이버
Microsoft Visual FoxPro에 데이터베이스를 구성 하 고 응용 프로그램 개발을 위한 강력한 개체 지향적 환경입니다. Microsoft Visual FoxPro ODBC 드라이버 응용 프로그램을를 열고, 쿼리, Visual FoxPro 및 데이터베이스 연결 열기 (ODBC) 인터페이스를 통해 FoxPro의 이전 버전에서 데이터를 업데이트할 수 있습니다.  
  
 예를 들어, Microsoft Visual FoxPro ODBC 드라이버를 사용 하 여 수행할 수 있습니다 다음.  
  
-   Microsoft Excel 워크시트에서 Visual FoxPro 데이터를 쿼리하고 업데이트 하려면 Microsoft 쿼리를 사용 합니다.  
  
-   Visual FoxPro 데이터를 사용 하 여 Microsoft Word를 사용 하 여 편지 병합 문자를 만듭니다.  
  
-   쿼리 및 Visual FoxPro 보기 및 Microsoft Access에서 테이블을 업데이트 합니다.  
  
-   Visual FoxPro Microsoft Visual Basic, Microsoft Visual c + + 및 C 응용 프로그램에 대 한 데이터 저장소로 사용 합니다.  
  
 다른 많은 작업을 수행 하는 드라이버를 사용할 수 있습니다. 다음 표에서 시작 하는 데 몇 가지 항목을 나열 합니다.  
  
|수행할 작업|참조 항목|  
|--------|---------|  
|Visual FoxPro 데이터를 사용 하 여 Microsoft Office를 사용 하 여 자세히 알아보기|[Microsoft Office에서 Visual FoxPro 데이터에 액세스](../../odbc/microsoft/accessing-visual-foxpro-data-from-microsoft-office.md)|  
|Visual FoxPro 데이터를 사용 하 여 Visual Basic 응용 프로그램에 대해 알아봅니다|[Visual FoxPro ODBC 드라이버를 사용 하 여 Visual Basic 응용 프로그램을 사용 하 여](../../odbc/microsoft/using-the-vfp-foxpro-odbc-driver-with-your-visual-basic-application.md)|
