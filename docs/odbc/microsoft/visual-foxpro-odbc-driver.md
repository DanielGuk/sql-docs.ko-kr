# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC 드라이버
Microsoft Visual FoxPro에 데이터베이스 생성 및 응용 프로그램 개발을 위한 강력한 개체 지향 환경입니다. Microsoft Visual FoxPro ODBC 드라이버 응용 프로그램을을 열고, 쿼리, Visual FoxPro 및 데이터베이스 연결 열기 (ODBC) 인터페이스를 통해 FoxPro의 이전 버전에서 데이터를 업데이트할 수 있습니다.  
  
 예를 들어 Microsoft Visual FoxPro ODBC 드라이버를 사용 하면 다음을 수행할 수 있습니다.:  
  
-   Microsoft Excel 워크시트에서 데이터를 쿼리하고 업데이트 Visual FoxPro Microsoft 쿼리를 사용 합니다.  
  
-   Visual FoxPro 데이터를 사용 하 여 Microsoft Word와 편지 병합 문자를 만듭니다.  
  
-   쿼리하고 Visual FoxPro 뷰 및 Microsoft Access에서 테이블을 업데이트 합니다.  
  
-   Visual FoxPro Microsoft Visual Basic, Microsoft Visual c + + 및 C 응용 프로그램에 대 한 데이터 저장소로 사용 합니다.  
  
 다양 한 작업을 수행 하기 위해 드라이버를 사용할 수 있습니다. 다음 표에서 시작 하는 데는 도움이 되는 몇 가지 항목을 나열 합니다.  
  
|수행할 작업|참조 항목|  
|--------|---------|  
|Visual FoxPro 데이터를 사용 하 여 Microsoft Office와 함께 하는 방법에 대 한 자세한 내용을 알아보려면|[Microsoft Office에서 Visual FoxPro 데이터에 액세스](../../odbc/microsoft/accessing-visual-foxpro-data-from-microsoft-office.md)|  
|Visual Basic 응용 프로그램의 Visual FoxPro 데이터를 사용 하는 방법에 대 한 자세한 내용은|[Visual Basic 응용 프로그램과 함께 Visual FoxPro ODBC 드라이버를 사용 하 여](../../odbc/microsoft/using-the-vfp-foxpro-odbc-driver-with-your-visual-basic-application.md)|  
|Visual c + +를 사용 하 여 Visual FoxPro 데이터에 액세스 하는 간단한 예제를 보려면|[Visual FoxPro ODBC 드라이버를 사용 하 여 C 또는 c + + 응용 프로그램](../../odbc/microsoft/using-the-visual-foxpro-odbc-driver-with-your-c-or-visual-c-application.md)|  
|지원 되는 하드웨어 및 소프트웨어의 목록을 보려면|[시스템 요구 사항](../../odbc/microsoft/system-requirements-visual-foxpro-odbc-driver.md)|  
  
 다운로드 하 고 Visual FoxPro ODBC 드라이버를 설치 하는 방법에 대 한 정보를 참조 하십시오. [Visual FoxPro ODBC 드라이버](http://go.microsoft.com/fwlink/?LinkId=121318) 에서 MSDN 다운로드 합니다.
