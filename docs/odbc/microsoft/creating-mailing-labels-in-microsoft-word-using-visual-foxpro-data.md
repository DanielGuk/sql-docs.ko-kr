# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>우편물 레이블 Visual FoxPro 데이터를 사용 하 여 Microsoft Word에서 만들기
Windows 95 또는 Windows 98 문서에 대 한 Microsoft Word에서 Visual FoxPro 데이터를 사용할 수 있습니다. 예를 들어 Visual FoxPro 테이블에 저장 된 고객 정보에서 우편물 레이블을 만들려고 할 수 있습니다.  
  
### <a name="to-create-mailing-labels"></a>우편물 레이블 만들기를  
  
1.  Microsoft Word에서 새 문서를 만듭니다.  
  
2.  도구 메뉴에서 편지 병합을 선택 합니다.  
  
3.  편지 병합 도우미에서 만들기를 선택 하 고 우편물 레이블을 선택 합니다.  
  
4.  주 문서에서 활성 창을 선택 합니다.  
  
5.  데이터 원본에서 데이터 가져오기를 선택 하 고 데이터 원본 열기를 선택 합니다.  
  
6.  데이터 원본 열기 대화 상자에서 MS 쿼리를 선택 합니다.  
  
7.  데이터 원본 선택 대화 상자에서 Visual FoxPro 데이터 원본을 선택한 다음 사용을 클릭 합니다.  
  
8.  데이터 원본에서 액세스 한 데이터베이스 테이블을 포함 하는 경우 테이블 추가 대화 상자에서 테이블을 선택 합니다. Microsoft Query 쿼리 디자이너의 위쪽 절반에 추가 된 테이블을 표시합니다.  
  
9. 에서 끌어서 테이블 아래에는 디자이너의 절반 쿼리에 대 한 필드를 선택 합니다.  
  
10. 파일 메뉴에서 Microsoft Word로 반환 데이터를 선택 합니다. Microsoft 쿼리 닫히고 선택한 데이터는 편지 병합 문서에서 사용할 수 있습니다.  
  
11. 주 문서에서 설치 프로그램을 선택 합니다.  
  
12. 레이블 옵션 대화 상자에서 프린터 및 레이블 정보 하 고 확인을 클릭 한 다음를 선택 합니다.  
  
13. 레이블 만들기 대화 상자에서 다음 확인을 클릭 하 고 우편물 레이블에 인쇄 하려는 필드를 선택 합니다.  
  
14. 편지 병합 도우미에서 문서를 사용 하 여 데이터 병합 병합을 클릭 합니다.  
  
15. 병합 대화 상자에서 병합을 클릭 한 다음 원하는 옵션을 선택 합니다.
