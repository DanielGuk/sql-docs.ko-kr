### <a name="to-run-this-application"></a>이 응용 프로그램을 실행 하려면  
  
1.  Microsoft SQL Server 실행 되 고 있는지 확인 합니다. 클릭 **시작**, 가리킨 **프로그램**, 가리킨 **Microsoft SQL Server 7.0**, 클릭 하 고 **Service Manager**합니다. 흰색 원에 녹색 화살표가 있으면 SQL Server가 실행 됩니다. 없는 경우 (됩니다 빨간색 사각형이 흰색 원 안에)를 클릭 하 여 **시작/계속**합니다.  
  
2.  Internet Explorer 4.0 이상 버전에서는 다음 주소를 입력 합니다.  
  
     **http://** *webserver* **/RDS/AddressBook/AddrBook.asp**  
  
     여기서 *웹 서버* RDS 서버 구성 요소 설치 되어 있는 웹 서버 이름입니다.  
  
3.  다음 제목이 "프로그램 관리자," 모든 사용자를 나열가 자신의 전자 메일 이름에 따라 사용자를 검색 하거나 기존 레코드를 편집 같이 주소록 샘플 응용 프로그램에서 다양 한 시나리오를 시도할 수 있습니다. 클릭 **찾을** 데이터 표에 사용 가능한 모든 이름으로 채울 수 있습니다.
