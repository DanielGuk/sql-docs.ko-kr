# <a name="connecting-to-data-sources"></a>데이터 원본에 연결
ADO **연결** 개체 DBMS, 파일 저장소 또는 쉼표로 구분 된 텍스트 파일을 포함 하 여 데이터 원본과 고유 세션을 나타냅니다. 클라이언트/서버 데이터베이스 시스템의 경우 ADO 연결 서버에는 실제 네트워크 연결 될 수 있습니다.  
  
 **연결** 개체가 지 원하는 다양 한 [속성 및 메서드](../../../ado/reference/ado-api/connection-object-properties-methods-and-events.md) 연결 구성을 지정 하기 위한 열기 및 연결을 닫는, 만들기 및 실행 명령을 데이터 원본에 대해 등의 데이터 원본 스키마 행 집합의 형태로 디자인에 대 한 정보를 제공 하 고 있습니다. 공급자, 일부 컬렉션, 메서드 또는 속성을 지 원하는 기능에 따라 한 **연결** 개체를 사용할 수 없습니다.  
  
 사용 하 여 데이터 원본에 연결할 수는 **연결** 또는 개체를 사용 하 여 한 **레코드 집합** 개체입니다.
