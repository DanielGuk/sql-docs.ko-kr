# <a name="appendix-d-ado-samples-in-the-windows-sdk"></a>Windows SDK의에서 부록 d: ADO 샘플
Windows SDK ADO 및 RDS 코드의 사용을 보여 주는 몇 가지 샘플 응용 프로그램을 포함 합니다. 이러한 샘플 ADO와 RDS를 사용 하는 방법을 배울 수 있도록 수는 있지만 프로덕션 서버에 배포할 수는 없습니다.

> [!NOTE]
>  회사, 이름 및 샘플에 사용 되는 데이터는 다른 설명이 없는 한 실제 데이터가 아닙니다.

 ADO 샘플을 찾으려면를 C:\Program Files\Microsoft SDKs\Windows\v6.0 같은 Windows SDK 루트 폴더로 이동 합니다. 그런 다음 Samples\dataaccess\ADO 하위 폴더로 이동 합니다.

 ADO 및 RDS 샘플 응용 프로그램 사용을 시작 하려면 ADO 및 RDS 샘플 폴더에서 default.htm을 각각 엽니다. 이러한 파일에는 각 사용 가능한 샘플 응용 프로그램에 설명 및 구성 하 고 샘플 소스 코드를 탐색 하는 방법 샘플을 실행 하는 방법을 설명 합니다.

 또한 ADO 문서에는이 온라인 도움말 항목으로 포함 되는 여러 언어로 코드 예제가 들어 있습니다. 이러한 예제에 대 한 자세한 내용은 참조 하세요. [ADO 코드 예제](../../../ado/reference/ado-api/ado-code-examples.md)합니다.
