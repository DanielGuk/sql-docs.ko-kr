# <a name="catalog-object-ado-md"></a>카탈로그 개체 (ADO MD)
다차원 데이터 공급자 (MDP)에 적용 되는 다차원 스키마 정보 (큐브 기본 차원, 계층, 수준 및 멤버)를 포함합니다.  
  
## <a name="remarks"></a>주의  
 컬렉션과의 속성을 사용 하 여 한 **카탈로그** 개체를 다음을 수행할 수 있습니다.  
  
-   설정 하 여 카탈로그를 열고는 [ActiveConnection](../../../ado/reference/ado-md-api/activeconnection-property-ado-md.md) 속성을 표준 ADO [연결](../../../ado/reference/ado-api/connection-object-ado.md) 개체 또는 유효한 연결 문자열입니다.  
  
-   식별 된 **카탈로그** 와 [이름](../../../ado/reference/ado-md-api/name-property-ado-md.md) 속성입니다.  
  
-   큐브를 사용 하 여 카탈로그에서 반복할는 [CubeDefs](../../../ado/reference/ado-md-api/cubedefs-collection-ado-md.md) 컬렉션입니다.  
  
 이 섹션에는 다음 항목 포함 되어 있습니다.  
  
-   [속성, 메서드 및 이벤트](../../../ado/reference/ado-md-api/catalog-object-properties-methods-and-events-ado-md.md)
