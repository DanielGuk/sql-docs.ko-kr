# <a name="ordinal-property-ado-md-position"></a>서 수 속성 (ADO MD 위치)
고유 하 게 식별 한 [위치](../../../ado/reference/ado-md-api/position-object-ado-md.md) 축을 따라 합니다.  
  
## <a name="return-values"></a>반환 값  
 반환 된 **긴** 정수 읽기 전용입니다.  
  
## <a name="remarks"></a>주의  
 **서 수** 속성은 [위치](../../../ado/reference/ado-md-api/position-object-ado-md.md) 개체의 인덱스에 해당는 **위치** 내에서 [위치](../../../ado/reference/ado-md-api/positions-collection-ado-md.md) 컬렉션입니다.  
  
 셀 신속 하 게 검색할 수 있습니다를 사용 하는 **서 수** 의 **위치** 로 각 축을 따라는 [항목](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md) 의 속성은 [셀 집합](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) 개체 .
