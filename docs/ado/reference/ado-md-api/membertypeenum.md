# <a name="membertypeenum"></a>MemberTypeEnum
에 대 한 설정을 지정 하는 [형식](../../../ado/reference/ado-md-api/type-property-ado-md.md) 속성의는 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 개체입니다.  
  
|상수|값|Description|  
|--------------|-----------|-----------------|  
|**adMemberAll**|4|나타냅니다는 **멤버** 개체는 해당 수준의 모든 멤버를 나타냅니다.|  
|**adMemberFormula**|3|나타냅니다는 **멤버** 개체 수식 식을 사용 하 여 계산 됩니다.|  
|**adMemberMeasure**|2|나타냅니다는 **멤버** 개체의 측정값 차원에 속하고 정량 특성을 나타냅니다.|  
|**adMemberRegular**|1.|기본. 나타냅니다는 **멤버** 개체 비즈니스 엔터티 인스턴스를 나타냅니다.|  
|**adMemberUnknown**|0|멤버의 유형을 확인할 수 없습니다.|
