# <a name="type-property-ado-md"></a>Type 속성 (ADO MD)
현재 형식을 나타내는 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md)합니다.  
  
## <a name="return-values"></a>반환 값  
 반환 된 [MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md) 값 및 읽기 전용입니다.  
  
## <a name="remarks"></a>주의  
 이 속성은의 경우에 지원 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 에 속하는 개체는 [수준](../../../ado/reference/ado-md-api/level-object-ado-md.md) 개체입니다. 이 속성에서 참조 되는 동안 오류가 발생 **멤버** 에 속하는 개체는 [위치](../../../ado/reference/ado-md-api/position-object-ado-md.md) 개체입니다.
