# <a name="leveldepth-property-ado-md"></a>ADO MD LevelDepth 속성
계층의 루트 사이 있는 수준 수를 나타내는 및 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md)합니다.  
  
## <a name="return-values"></a>반환 값  
 반환은 **긴** 정수, 읽기 전용입니다.  
  
## <a name="remarks"></a>Remarks  
 사용 하 여는 **LevelDepth** 속성의 거리를 확인 하는 [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md)계층의 루트 수준에서 개체입니다. **LevelDepth**루트에 있는 멤버의 수준은 0입니다. 이에 해당 하는 [깊이](../../../ado/reference/ado-md-api/depth-property-ado-md.md) 속성의는 [수준](../../../ado/reference/ado-md-api/level-object-ado-md.md) 개체입니다.
