# <a name="type-property-table-adox"></a>Type 속성 (테이블) (ADOX)
테이블의 유형을 나타냅니다.  
  
## <a name="return-values"></a>반환 값  
 반환 된 **문자열** ; 예: "TABLE", "SYSTEM TABLE" 또는 "전역 임시"에 대 한 테이블의 형식을 지정 하는 값입니다.  
  
## <a name="remarks"></a>주의  
 이 속성은 읽기 전용입니다.
