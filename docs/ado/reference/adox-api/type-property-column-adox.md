# <a name="type-property-column-adox"></a>Type 속성 (열) (ADOX)
데이터 형식을 열을 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 설정 하거나 반환는 **긴** 값 중 하나일 수 있습니다는 [DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md) 상수입니다. 기본값은 **adVarWChar**합니다.  
  
## <a name="remarks"></a>주의  
 이 속성은 읽기/쓰기 될 때까지 [열](../../../ado/reference/adox-api/column-object-adox.md) 개체 컬렉션이 나 그 이후에 읽기 전용으로 설정 하는 다른 개체에 추가 됩니다.
