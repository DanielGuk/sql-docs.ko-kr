# <a name="unique-property-adox"></a>고유한 속성 (ADOX)
인덱스 키를 고유 해야 하는지 여부를 나타냅니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 설정 하 고 반환 된 **부울** 값입니다.  
  
## <a name="remarks"></a>주의  
 기본값은 **False**입니다.  
  
 이 속성은 읽기 전용 [인덱스](../../../ado/reference/adox-api/index-object-adox.md) 컬렉션에 이미 추가 된 개체입니다.
