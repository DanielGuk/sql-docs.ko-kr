# <a name="actionenum"></a>ActionEnum
때 수행할 동작의 유형을 지정 [SetPermissions](../../../ado/reference/adox-api/setpermissions-method-adox.md) 호출 됩니다.  
  
|상수|값|Description|  
|--------------|-----------|-----------------|  
|**adAccessDeny**|3|그룹 또는 사용자의 특정된 사용 권한이 거부 됩니다.|  
|**adAccessGrant**|1|그룹 또는 사용자 이상는 요청 된 사용 권한을 갖습니다.|  
|**adAccessRevoke**|4|그룹이 나 사용자의 모든 명시적 액세스 권한이 해지 됩니다.|  
|**adAccessSet**|2|그룹 또는 사용자 요청 된 사용 권한을 갖습니다.|
