# <a name="ruleenum"></a>RuleEnum
시 따라야 하는 규칙 지정는 [키](../../../ado/reference/adox-api/key-object-adox.md) 삭제 됩니다.  
  
|상수|값|Description|  
|--------------|-----------|-----------------|  
|**adRICascade**|1.|변경 내용을 모두 합니다.|  
|**adRINone**|0|기본. 아무 동작도 수행되지 않습니다.|  
|**adRISetDefault**|3|외래 키 값을 기본값으로 설정 됩니다.|  
|**adRISetNull**|2|외래 키 값이 설정 되어 null로 합니다.|
