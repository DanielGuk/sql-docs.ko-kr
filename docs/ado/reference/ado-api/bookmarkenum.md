# <a name="bookmarkenum"></a>BookmarkEnum
작업을 시작할 위치를 나타내는 책갈피를 지정 합니다.  
  
|상수|값|Description|  
|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|현재 레코드에서 시작 합니다.|  
|**adBookmarkFirst**|1.|첫 번째 레코드에서 시작 합니다.|  
|**adBookmarkLast**|2|마지막 레코드에서 시작 합니다.|
