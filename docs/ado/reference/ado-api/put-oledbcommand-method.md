# <a name="putoledbcommand-method"></a>put_OLEDBCommand 메서드
이 메서드는 아무런 작업도 수행 하 고 항상 S_OK를 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
HRESULT put_OLEDBCommand(  
      IUnknown *pOLEDBCommand  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 *pOLEDBCommand*  
 [in] OLE DB 명령 개체에 대 한 포인터입니다.
