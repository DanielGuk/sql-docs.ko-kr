# R Server (Standalone)
  **Microsoft R 서버 (독립 실행형)** Microsoft R 서비스에서 제공 하는 엔터프라이즈 수준의 분석 플랫폼의 일부입니다.  
  
 R 지원, 확장성 및 보안을 제공 하 여 확장 합니다. [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] 병렬을 추가 하 고 사용자가 엔터프라이즈 규모에서 데이터를 분석 하 고 메모리에 들어갈 수 있는 것 보다 훨씬 더 큰 데이터를 사용할 수 있도록 데이터의 청크 분할된 처리 오픈 소스 R의 메모리 제한 사항을 해결 합니다.  
  
  
## Microsoft R Server 시작  
  
 R 응용 프로그램을 지원 하기 위한 Windows server를 설정 하는 방법에 대 한 자세한 내용은 참조 [독립 실행형 R 서버 만들기](../../advanced-analytics/r-services/create-a-standalone-r-server.md)합니다.  

응용 프로그램 내의 R analytics를 포함 하 고 R 백 엔드 시스템과 통합 하기 위한 DeployR, 함께 R 서버 (독립 실행형)를 포함 하는 Azure 가상 컴퓨터를 제공할 수 있습니다. 참조 항목:

+ [R 서버 가상 컴퓨터를 프로 비전](../../advanced-analytics/r-services/provision-the-r-server-only-sql-server-2016-enterprise-vm-on-azure.md)
+ [R 배포](https://msdn.microsoft.com/microsoft-r/deployr-about)
  
샘플, 자습서 및 Microsoft R에 대 한 자세한 내용은 참조 하십시오 [Microsoft R 제품](https://msdn.microsoft.com/microsoft-r/microsoft-r-getting-started)합니다.
