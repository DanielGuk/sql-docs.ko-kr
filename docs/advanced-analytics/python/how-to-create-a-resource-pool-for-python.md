# <a name="how-to-create-a-resource-pool-for-python"></a>Python에 대 한 리소스 풀을 만드는 방법
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

이 섹션은 개발입니다.

> [!NOTE]
> Python 지원은 SQL Server 2017의 새로운 기능 하며 시험판에서 있습니다. 자세한 내용은 곧 찾습니다.

일반적으로 Python을 실행 하는 하나를 포함 하 여, SQL Server 2016에서 R 스크립트의 리소스 관리에 대해 제공 된 동일한 프레임 워크를 사용 하 여 모든 외부 스크립트를 모니터링할 수 있습니다. 자세한 내용은 참조 [R에 대 한 리소스 관리](../r/resource-governance-for-r-services.md)합니다.
