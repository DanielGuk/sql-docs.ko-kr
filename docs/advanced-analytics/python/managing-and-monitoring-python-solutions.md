# <a name="monitoring"></a>모니터링


> [!NOTE]
> Python 지원은 SQL Server 2017의 새로운 기능 하며 시험판에서 있습니다.

Python에 대 한 지원은 SQL Server, SQL Server 2017 CTP 2.0에서 사용할 수 있는의 새로운 기능 합니다.

그러나 외부 스크립트 실행 및 데이터 보안에 대 한 전체 프레임 워크는 오른쪽의 경우와 동일 Python을 실행 하는 하나를 포함 하 여, SQL Server 2016에서 R 스크립트 실행에 대해 제공 된 동일한 프레임 워크를 사용 하 여 모든 외부 스크립트를 모니터링할 수 있습니다. 자세한 내용은 참조 [Monitoring R Solutions](../r/managing-and-monitoring-r-solutions.md)합니다.
