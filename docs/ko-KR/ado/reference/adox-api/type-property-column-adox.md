## <a name="settings-and-return-values"></a>설정 및 반환 값  
 설정 하거나 반환는 **긴** 값 중 하나일 수 있습니다는 [DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md) 상수입니다. 기본값은 **adVarWChar**합니다.
