|--------------|-----------|-----------------|

|**adObjectTypeLevel**|3|A [수준](../../../ado/reference/ado-md-api/level-object-ado-md.md) 개체입니다.|  
|**adObjectTypeMember**|4|A [멤버](../../../ado/reference/ado-md-api/member-object-ado-md.md) 개체입니다.|
