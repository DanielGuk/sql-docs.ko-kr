# <a name="redistributing-the-microsoft-jdbc-driver"></a>Microsoft JDBC Driver 재배포
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

## <a name="redistribute-the-microsoft-jdbc-41-42-60-62-and-64-driver"></a>4.1, 4.2, 6.0, 6.2, 및 6.4 Microsoft JDBC Driver를 재배포
JDBC Driver 4.1, 4.2, 6.0, 6.2, 및 6.4 재배포할 수 있습니다. 4.1, 4.2, 6.0에 대 한 사용권 계약에서 "배포 가능 코드" 절을 검토 하십시오 6.2 및 6.4 버전의 드라이버입니다.
    
## <a name="register-to-redistribute-the-microsoft-jdbc-40-driver"></a>Microsoft JDBC 4.0 Driver를 재배포하도록 등록  
 Microsoft JDBC 4.0 Driver 재배포 하면 전에 등록이 필요 합니다. 4.0 버전의 드라이버에 대 한 사용권 계약을 검토 하려면 다음 링크를 클릭 합니다.  원할 경우 기록을 위해 사용권 계약의 사본을 인쇄하여 보관할 수 있습니다.  
  
 사용권 계약에 동의하면 등록 페이지와 다운로드 페이지로 차례로 이동됩니다.  
  
 [Microsoft JDBC Driver 4.0 for SQL Server에 대 한 재배포 라이선스](https://msdn.microsoft.com/sqlserver/jj589698)  
  
 참고: Microsoft 보안 인증 및 등록에 대 한 Microsoft 계정을 사용합니다. 을 등록 하면 재배포 권한을 대 한 SQL Server JDBC Driver 재배포에 대 한 Microsoft 계정을 통해 하면 보안 프로필의 일부분이 되는 정보를 제공 하 라는 메시지가 표시 됩니다.  
  
 가져올 수 없는 하나 이미, 하는 경우는 [Microsoft 계정](https://signup.live.com/)
