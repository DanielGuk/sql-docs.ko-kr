# <a name="diagnosing-problems-with-the-jdbc-driver"></a>JDBC 드라이버 관련 문제 진단
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  응용 프로그램의 디자인 및 개발 정도에 관계없이 문제는 발생합니다. 문제가 발생하면 이러한 문제를 진단하는 기술을 보유하는 것이 중요합니다. 사용 하는 경우는 [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)], 발생할 수 있는 몇 가지 일반적인 문제는 올바른 드라이버 버전이 말거나 데이터베이스에 연결 하지 못합니다.  
  
 이 섹션의 항목에서는 오류 처리, 드라이버 버전 확인, 연결성 문제 추적 및 해결을 포함하여 이러한 문제 및 다른 문제를 진단하는 여러 기술에 대해 설명합니다.
