# <a name="getbigdecimal-method-sqlservercallablestatement"></a>getBigDecimal 메서드(SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 매개 변수의 값을 검색하여 java.math.BigDecimal로 반환합니다.  
  
## <a name="overload-list"></a>오버로드 목록  
  
|이름|Description|  
|----------|-----------------|  
|[getBigDecimal (int)](../../../connect/jdbc/reference/getbigdecimal-method-int.md)|매개 변수 인덱스가 지정된 경우 지정된 매개 변수의 값을 검색하여 전체 자릿수를 모두 표시하는 java.math.BigDecimal로 반환합니다.|  
|[getBigDecimal (int, int)](../../../connect/jdbc/reference/getbigdecimal-method-int-int.md)|(사용되지 않음) 매개 변수 인덱스와 소수 자릿수가 지정된 경우 지정된 매개 변수의 값을 검색하여 java.math.BigDecimal로 반환합니다.|  
|[getBigDecimal (java.lang.String)](../../../connect/jdbc/reference/getbigdecimal-method-java-lang-string.md)|매개 변수 이름이 지정된 경우 지정된 매개 변수의 값을 검색하여 전체 자릿수를 모두 표시하는 java.math.BigDecimal로 반환합니다.|  
|[getBigDecimal (java.lang.String, int)](../../../connect/jdbc/reference/getbigdecimal-method-java-lang-string-int.md)|(사용되지 않음) 매개 변수 이름과 소수 자릿수가 지정된 경우 지정된 매개 변수의 값을 검색하여 java.math.BigDecimal로 반환합니다.|
