# <a name="isnamed-method-sqlserversavepoint"></a>isNamed 메서드(SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  저장점의 이름이 지정되었는지 확인합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public boolean isNamed()  
```  
  
## <a name="return-value"></a>반환 값  
 **true** 저장 점의 이름이 지정 되었으면이 있습니다. 그렇지 않으면 **false**입니다.
