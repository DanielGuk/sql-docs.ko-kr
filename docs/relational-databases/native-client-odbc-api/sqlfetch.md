# <a name="sqlfetch"></a>SQLFetch
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  향상 된 날짜 및 시간 기능을 SQLFetch 지원 합니다.  
  
 날짜/시간 형식의 결과 열 값이 변환 되는 방법에 대 한 정보를 참조 하십시오. [SQL에서 C로 변환을](../../relational-databases/native-client-odbc-date-time/datetime-data-type-conversions-from-sql-to-c.md)합니다.  
  
## <a name="sqlfetch-support-for-large-clr-udts"></a>큰 CLR UDT에 대한 SQLFetch 지원  
 **SQLFetch** 는 큰 CLR UDT(사용자 정의 형식)를 지원합니다. 자세한 내용은 참조 [Large CLR User-Defined 형식 & #40; ODBC & #41;](../../relational-databases/native-client/odbc/large-clr-user-defined-types-odbc.md)합니다.
