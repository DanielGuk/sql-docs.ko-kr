# <a name="description-property-sqlservice-class"></a>Description 속성(SqlService 클래스)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]참조 된 서비스에 대 한 설명을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object.Description [= value]  
```  
  
## <a name="parts"></a>부분  
 *개체*  
 서비스를 나타내는 [SqlService 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 참조된 서비스에 대한 텍스트 설명을 포함하는 문자열 값입니다.
