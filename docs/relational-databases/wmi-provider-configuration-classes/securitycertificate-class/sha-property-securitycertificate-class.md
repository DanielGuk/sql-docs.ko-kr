# <a name="sha-property-securitycertificate-class"></a>SHA 속성(SecurityCertificate 클래스)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  보안 인증서의 SHA 지문 특성을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object.SHA [= value]  
```  
  
## <a name="parts"></a>부분  
 *개체*  
 보안 인증서를 나타내는 [SecurityCertificate 클래스](../../../relational-databases/wmi-provider-configuration-classes/securitycertificate-class/securitycertificate-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 보안 인증서의 SHA 지문 특성을 지정하는 문자열 값입니다.
