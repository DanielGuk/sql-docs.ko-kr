# <a name="sqlserveralias-class"></a>SqlServerAlias 클래스
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [SqlServerAlias 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlserveralias-class/sqlserveralias-class.md) 는 서버 연결 별칭을 나타냅니다.  
  
 서버 연결 별칭은 다음 두 가지 모두에 해당하는 경우 필요합니다.  
  
-   클라이언트가 기본 네트워크 전송이 아닌 네트워크 전송을 통해 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 인스턴스에 연결하는 경우  
  
-   클라이언트가 대체 명명된 파이프에서 수신하는 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 인스턴스에 연결하는 경우  
  
 **참고:** [SqlServerAlias 클래스](../../../relational-databases/wmi-provider-configuration-classes/sqlserveralias-class/sqlserveralias-class.md) 는 Provider 클래스에서 **Put** 메서드를 상속하지만 **Provider::Put** 메서드가 표시하는 결과를 반환하지는 않습니다. 자세한 내용은 WMI 설명서를 참조하십시오.
