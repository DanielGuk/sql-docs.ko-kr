# <a name="device-not-ready-error"></a>장치 준비 안 됨 오류
  이 규칙은 시스템 이벤트 로그에서 EventId 15를 검사합니다. 이 오류는 SCSI 호스트 어댑터 구성 문제 또는 이와 관련된 문제로 인해 발생할 수 있습니다.

 이 오류를 해결하는 방법은 다음 Microsoft 기술 자료 문서를 참조하십시오.
