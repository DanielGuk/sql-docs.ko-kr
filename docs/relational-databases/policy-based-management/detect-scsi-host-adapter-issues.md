# SCSI 호스트 어댑터 문제 검색
  이 규칙은 시스템 이벤트 로그에서 EventId 1066을 검사합니다. 이 오류는 SCSI 호스트 어댑터 구성 문제 또는 장치 오작동으로 인해 발생합니다.  
  
## 최선의 구현 방법 권장 사항  
 이 오류를 해결하는 방법은 다음 Microsoft 기술 자료 문서를 참조하십시오.  
  
-   [Microsoft 기술 자료 문서 311081](http://go.microsoft.com/fwlink/?linkid=117744)
