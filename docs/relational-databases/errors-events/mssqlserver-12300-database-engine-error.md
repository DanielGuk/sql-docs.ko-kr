# <a name="mssqlserver12300"></a>MSSQLSERVER_12300
  
## <a name="details"></a>세부 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|12300|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|메시지 텍스트|계산 열은 '*construct*'에서 지원되지 않습니다.|
