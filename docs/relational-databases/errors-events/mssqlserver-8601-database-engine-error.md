# <a name="mssqlserver8601"></a>MSSQLSERVER_8601
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>세부 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|8601|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|INVALID_INTERFACE_ERR|  
|메시지 텍스트|내부 쿼리 프로세서 오류: 쿼리 프로세서에서 필요한 인터페이스에 대한 액세스 권한을 얻을 수 없습니다.|  
  
## <a name="explanation"></a>설명  
내부 처리 오류입니다. 필요한 인터페이스에 대한 액세스 권한을 얻을 수 없습니다.  
  
## <a name="user-action"></a>사용자 동작  
주 서비스 공급자에게 이 오류 메시지의 모든 정보를 제공하고 문의하십시오.
