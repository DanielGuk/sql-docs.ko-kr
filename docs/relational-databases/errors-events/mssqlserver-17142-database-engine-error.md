# <a name="mssqlserver17142"></a>MSSQLSERVER_17142
  
## <a name="details"></a>세부 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|17142|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|INIT_SRVC_PAUSED|  
|메시지 텍스트|SQL Server 서비스가 일시 중지되었습니다. 새 연결은 허용되지 않습니다. 서비스를 다시 시작하려면 SQL 컴퓨터 관리자 또는 제어판의 서비스 응용 프로그램을 사용하십시오.|  
  
## <a name="explanation"></a>설명  
서비스 제어 관리자에 의해 SQL Server 서비스가 일시 중지되었습니다.  
  
## <a name="user-action"></a>사용자 동작  
SQL Server 구성 관리자를 사용하여 SQL Server 서비스를 재개합니다.
