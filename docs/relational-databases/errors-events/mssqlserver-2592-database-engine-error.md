|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|2592|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|DBCC_REPAIR_INDEX_REBUILT|
