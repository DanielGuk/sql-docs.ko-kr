## <a name="details"></a>세부 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|2592|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|DBCC_REPAIR_INDEX_REBUILT|  
|메시지 텍스트|복구: 데이터베이스 'DATABASE'에서 개체 OBJECT의 INDEX_TYPE 인덱스가 다시 작성되었습니다.|
