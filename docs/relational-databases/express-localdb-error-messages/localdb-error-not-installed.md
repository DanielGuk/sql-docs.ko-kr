# <a name="localdberrornotinstalled"></a>LOCALDB_ERROR_NOT_INSTALLED
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## <a name="details"></a>세부 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|278|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|참고: 메시지 텍스트가 비어 있습니다.,이 메시지 즉 전체 이므로 LocalDB API (HRESULTS를 메시지 텍스트에 매핑하는 FormatMessage 함수 포함)를 사용할 수 없습니다.|  
  
## <a name="explanation"></a>설명  
 로컬 데이터베이스 런타임이 컴퓨터에 설치되지 않았습니다.  
  
## <a name="user-action"></a>사용자 동작  
 로컬 데이터베이스 런타임을 설치하고 작업을 다시 시도하십시오.
