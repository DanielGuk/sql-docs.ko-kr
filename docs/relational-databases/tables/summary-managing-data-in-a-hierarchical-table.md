# 요약: 계층적 테이블의 데이터 관리
**hierarchyid** 데이터 형식을 사용하면 계층적 데이터를 반영하는 테이블을 쉽게 만든 다음 깊이 우선 및 너비 우선 인덱스를 만들어 테이블에 효율적으로 액세스할 수 있습니다. 계층 메서드를 사용하여 데이터를 쿼리하고 조작합니다.
