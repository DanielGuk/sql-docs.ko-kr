# 복제 에이전트
  복제는 "에이전트"라는 여러 개의 독립 실행형 프로그램을 사용하여 변경 내용 추적 및 데이터 배포와 연관된 태스크를 수행합니다. 이 설명서 섹션에는 다음 복제 에이전트에 대한 매개 변수 참조가 포함되어 있습니다.
