# 게시자 속성 - 게시자, 구독자
   **구독자** 의 페이지는 **게시자 속성** 의 버전을 실행 하는 게시자에 대 한 대화 상자를 사용 하는 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 이전에 [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]합니다. 이 페이지를 사용하여 구독자가 이 게시자의 게시에서 데이터를 받도록 설정할 수 있습니다. 구독자가 이 게시자에서 데이터를 받도록 설정해도 이 게시자의 게시에 대한 구독이 생성되지 않습니다. 구독을 만들려면 새 구독 마법사를 사용해야 합니다.  
  
## 옵션  
 **구독자**  
  **구독자** 속성 표 형식으로이 게시자의 게시에서 데이터를 받도록 설정 된 구독자를 표시 합니다. 속성 단추를 클릭 (**...**) 추가 속성 보기 및 설정에 대 한 구독자 옆에 있습니다.  
  
 **추가**  
 클릭 **추가** 구독자를 추가 하 고 클릭 한 다음 **SQL Server 구독자 추가** 또는 **비-SQL Server 구독자 추가**합니다.
