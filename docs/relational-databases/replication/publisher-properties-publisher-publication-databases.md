# 게시자 속성 - 게시자, 게시 데이터베이스
  **게시자 속성** 대화 상자의 **게시 데이터베이스** 페이지를 사용하여 **sysadmin** 고정 서버 역할의 사용자가 복제용 데이터베이스를 설정할 수 있습니다. 데이터베이스를 설정 하면 해당 데이터베이스에 게시 하지 않습니다. 대신에 모든 사용자를 수 있습니다는 **db_owner** 고정된 데이터베이스 역할에 데이터베이스에서 하나 이상의 게시를 만들 수 있습니다.  
  
## 옵션  
 **트랜잭션**  
 사용자에 게 허용 하려면이 확인란을 선택는 **db_owner** 고정된 데이터베이스 역할을 데이터베이스에 스냅숏 게시 또는 트랜잭션 게시를 만듭니다.  
  
 **병합**  
 사용자에 게 허용 하려면이 확인란을 선택 된 **db_owner** 병합 게시를 만들 데이터베이스의 고정된 데이터베이스 역할입니다.
