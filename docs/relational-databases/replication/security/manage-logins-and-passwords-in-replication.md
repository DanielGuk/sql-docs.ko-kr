# 복제의 로그인 및 암호 관리
  복제를 구성할 때 복제 에이전트에 대한 로그인과 암호를 지정합니다. 복제를 구성한 후에는 로그인과 암호를 변경할 수 있습니다. 자세한 내용은 [View and Modify Replication Security Settings](../../../relational-databases/replication/security/view-and-modify-replication-security-settings.md)을(를) 참조하세요. 복제 에이전트에서 사용 되는 계정에 대 한 암호를 변경 하면 실행 [sp_changereplicationserverpasswords (& a) #40; TRANSACT-SQL & #41;](../../../relational-databases/system-stored-procedures/sp-changereplicationserverpasswords-transact-sql.md)합니다.
