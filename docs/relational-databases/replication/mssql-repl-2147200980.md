# MSSQL_REPL-2147200980
    
## 메시지 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|-2147200980|  
|이벤트 원본|MSSQLServer|  
|심볼 이름||  
|메시지 텍스트|구독이 만료되었습니다. 구독을 다시 초기화하도록 표시하고 병합 에이전트를 다시 시작하여 구독을 다시 초기화하십시오.|  
  
## 설명  
 이 오류는 익명 구독이 만료되었을 때 발생합니다.  
  
## 사용자 동작  
 sp_reinitmergepullsubscription을 사용하여 익명 구독을 다시 초기화한 후 병합 작업을 다시 실행합니다.
