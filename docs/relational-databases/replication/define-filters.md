# 필터 정의
  **필터 정의** 대화 상자를 사용하여 필터를 정의한 다음 데이터 충돌에 적용하여 충돌의 하위 집합을 표로 볼 수 있습니다. 필터를 정의 하려면에서 연산자를 선택은 **연산자** 드롭다운 목록 상자 하 고 다음 값을 입력 합니다. 예를 들어 충돌 시 무시 되는 서버 충돌만 표시를 **ReplTest1**, 선택, **같지** 에서 **연산자** 드롭다운 목록 상자에 입력 한 **ReplTest1** 첫 번째에서 **값** 열입니다.  
  
## 옵션  
 **연산자**  
 필터에 대 한 연산자와 같은 선택 **보다 작거나**합니다.  
  
 **값**  
 필터 값을 입력합니다. 대부분의 연산자는 첫 번째 **값** 열에만 값을 입력하면 되지만 **사이** 및 **사이에 있지 않음** 연산자는 두 **값** 열에 값을 모두 입력해야 합니다.  
  
 **지우기**  
 이 단추를 클릭하여 이전에 정의한 필터를 모두 지울 수 있습니다.
