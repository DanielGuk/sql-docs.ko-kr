# MSSQL_REPL-2147199371
    
## 메시지 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|-2147199371|  
|이벤트 원본|MSSQLServer|  
|심볼 이름||  
|메시지 텍스트|IIS 서버로 보낸 요청이 4GB를 초과하므로 지원되지 않습니다. 'UploadGenerationsPerBatch' 매개 변수에 더 작은 값을 사용하십시오.|  
  
## 설명  
 웹 동기화를 사용할 경우 업로드된 메시지의 크기는 4GB보다 크지 않아야 합니다.  
  
## 사용자 동작  
 UploadGenerationsPerBatch 매개 변수의 값을 낮춥니다.
