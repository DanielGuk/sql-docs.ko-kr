# 배포자 암호
  경우에는 **게시자** 페이지가 마법사를 사용 하도록 설정한 하나 이상의 게시자를 원격 배포자로이 서버를 사용 하 여 게시자와 사용 하 여 원격 배포자 간 연결 복제에 대 한 암호를 사용 하면 지정 해야 합니다.는 **distributor_admin** 로그인 합니다. 새 게시 마법사 또는 배포 구성 마법사의 **관리 암호** 페이지에서 이 원격 배포자를 사용하는 각 게시자에 대해 동일한 암호를 입력해야 합니다. 배포자에 대 한 보안에 자세한 내용은 참조 [배포자 보안 설정](../../relational-databases/replication/security/secure-the-distributor.md)합니다.  
  
## 옵션  
 **암호**  
 게시자와 원격 배포자 간 연결에 대한 강력한 암호를 입력합니다.  
  
 **암호 확인**  
 암호를 다시 입력하여 올바로 입력되었는지 확인합니다.
