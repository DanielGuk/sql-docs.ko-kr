# 관리 암호
  경우에는 **배포자** 페이지를 원격 배포자를이 게시자에 대 한 선택 하면이 마법사를 사용 하 여 배포자 및 게시자 간의 연결 복제에서는 대 한 암호를 입력 해야는 **distributor_admin** 로그인 합니다. 입력하는 암호는 배포 구성 마법사의 **배포자 암호** 페이지 또는 **배포자 속성** 대화 상자의 **게시자** 페이지에서 지정한 암호와 일치해야 합니다.  
  
## 옵션  
 **암호**  
 게시자와 원격 배포자 간의 연결에 대한 암호를 입력합니다.  
  
 **암호 확인**  
 암호를 다시 입력하여 올바로 입력되었는지 확인합니다.
