# 지연 업데이트 충돌 해결 옵션 설정(SQL Server Management Studio)
  충돌 해결 옵션 지원에서 지연 업데이트 구독 하는 게시에 대 한 설정의 **구독 옵션** 의 페이지는 **게시 속성-\< 게시>** 대화 상자입니다. 이 대화 상자에 액세스하는 방법은 [View and Modify Publication Properties](../../../relational-databases/replication/publish/view-and-modify-publication-properties.md)을 참조하세요.  
  
### 지연 업데이트 충돌 해결 옵션을 설정하려면  
  
1.  에 **구독 옵션** 의 페이지는 **게시 속성-\< 게시>** 대화 상자에서 다음 중 하나에 대 한 값은 **충돌 해결 정책을** 옵션:  
  
    -   **게시자 변경 내용 유지**  
  
    -   **구독자 변경 내용 유지**  
  
    -   **구독 다시 초기화**
