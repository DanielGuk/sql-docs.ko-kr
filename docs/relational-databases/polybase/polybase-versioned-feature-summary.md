# <a name="polybase-versioned-feature-summary"></a>PolyBase 버전 기능 요약
[!INCLUDE[tsql-appliesto-ss2016-xxxx-asdw-pdw_md](../../includes/tsql-appliesto-ss2016-xxxx-asdw-pdw-md.md)]

  SQL Server 제품 및 서비스에 사용할 수 있는 PolyBase 기능 요약입니다.  
  
## <a name="feature-summary-for-product-releases"></a>제품 릴리스에 대한 기능 요약  
 이 표에서는 PolyBase 및 이를 사용할 수 있는 제품에 대한 주요 기능을 요약합니다.  
  
### [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)]  
 이러한 기능은 온-프레미스 또는 Azure 가상 컴퓨터에서 실행되는 [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] 에 적용됩니다.  PolyBase는 SQL Server 2014 이전 버전에서는 사용할 수 없습니다.  
  
|||  
|-|-|  
|**기능**|**가용성**|  
|다음을 사용하여 Hadoop 데이터 쿼리 [!INCLUDE[tsql](../../includes/tsql-md.md)]|예|  
|다음을 사용하여 Azure Blob 저장소 쿼리 [!INCLUDE[tsql](../../includes/tsql-md.md)]|예|  
|Hadoop에서 데이터 가져오기|예|  
|Azure Blob 저장소에서 데이터 가져오기|예| 
|Azure Data Lake Store에서 데이터 가져오기|아니요|   
|Hadoop으로 데이터 내보내기|예|  
|Azure Blob 저장소로 데이터 내보내기|예|  
|Azure Data Lake Store에서 데이터 내보내기|아니요|
|Microsoft의 BI 도구에서 PolyBase 쿼리 실행|예|  
|Hadoop으로 쿼리 계산 푸시다운|예|
