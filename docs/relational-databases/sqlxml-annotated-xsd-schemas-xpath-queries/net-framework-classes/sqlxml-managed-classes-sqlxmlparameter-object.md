# <a name="sqlxml-managed-classes---sqlxmlparameter-object"></a>SQLXML 관리 되는 클래스-SqlXmlParameter 개체
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
SqlXmlParameter 개체는 이러한 속성을 지원합니다.  
  
 이름  
 매개 변수의 이름입니다. 명령에 매개 변수가 전달될 수 있습니다. SqlXmlCommand 개체의 CreateParameter 메서드 호출 매개 변수 개체를 생성 합니다.  
  
 Value  
 매개 변수의 값입니다. 작업 예제를 참조 하세요. [SQL 쿼리 실행 &#40; SQLXML 관리 되는 클래스 &#41; ](../../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/net-framework-classes/executing-sql-queries-sqlxml-managed-classes.md).
