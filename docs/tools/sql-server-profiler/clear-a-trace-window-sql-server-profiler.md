# 추적 창 지우기(SQL Server Profiler)
  이 항목에서는 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]를 사용하여 실행 중인 추적의 추적 창을 지우는 방법을 설명합니다.  
  
### 추적 창을 지우려면  
  
1.  지울 추적 창을 선택합니다.  
  
2.  **편집**메뉴에서 **추적 창 지우기**를 클릭합니다.  
  
     추적 창의 내용이 제거됩니다. 추적 창 또는 테이블이 저장된 경우 해당 파일 또는 테이블의 내용이 변경되지 않은 채로 유지됩니다.
