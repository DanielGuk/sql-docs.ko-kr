# 일시 중지 또는 중지 후 추적 실행(SQL Server Profiler)
  이 항목에서는 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]를 사용하여 추적을 일시 중지하거나 중지한 후 추적을 다시 시작하는 방법에 대해 설명합니다.  
  
### 추적을 일시 중지하거나 중지한 후 추적을 실행하려면  
  
1.  중지 또는 일시 중지된 추적을 포함하는 창을 선택합니다.  
  
2.  **파일** 메뉴에서 **추적 실행**을 클릭합니다.
