# 요약: 솔루션 및 스크립트 프로젝트
[!INCLUDE[msCoName](../../includes/msconame-md.md)] Visual Studio의 광범위한 개발자 기능을 활용하여 Microsoft [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]는 엔터프라이즈급 코드 관리를 수행하려는 개발자에게 필요한 기능을 제공합니다. 사용자 지정 템플릿을 만들어 반복 태스크를 지원하고 중앙의 조정된 환경에서 복잡한 스크립트 프로젝트를 저장 및 관리할 수 있습니다.
