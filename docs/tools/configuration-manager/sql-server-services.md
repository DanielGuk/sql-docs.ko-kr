# <a name="sql-server-services"></a>SQL Server 서비스
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]이 섹션에 대 한 F1 도움말 항목이 포함 되어는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 서비스 대화 상자에 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Configuration Manager입니다.  
  
> [!NOTE]  
>  리소스를 호스트하지 않는 클러스터의 노드에서 **SQL Server 구성 관리자** 를 열면 서비스가 중지된 상태로 표시됩니다.  
  
> [!NOTE]  
>  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 서비스의 노드에 컴퓨터에서 사용 가능한 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 서비스가 채워져 있지 않으면 WMI(Windows Management Instrumentation)에 액세스할 수 있는 권한이 없는 것입니다. WMI에 대한 사용 권한을 구성하려면 온라인 설명서의 "방법: WMI를 구성하여 SQL Server 도구에 서버 상태 표시" 항목을 검색합니다.
