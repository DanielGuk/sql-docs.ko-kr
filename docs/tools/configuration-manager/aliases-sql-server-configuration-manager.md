# <a name="aliases-sql-server-configuration-manager"></a>별칭(SQL Server 구성 관리자)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]별칭은 연결을 사용할 수 있는 대체 이름입니다. 별칭은 연결 문자열의 필수 요소를 캡슐화하고 사용자가 선택한 이름으로 나타납니다. 이 컴퓨터의 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 클라이언트에 대한 별칭을 만들려면 콘솔 창에서 **별칭** 을 마우스 오른쪽 단추로 클릭한 다음 **새 별칭**을 클릭합니다. 이 컴퓨터의 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 클라이언트에 대한 기존 별칭을 구성하려면 콘솔 창에서 **별칭** 을 클릭하고 세부 정보 창에서 기존 별칭을 마우스 오른쪽 단추로 클릭한 다음 **속성**을 클릭합니다.
