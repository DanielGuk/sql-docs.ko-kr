# NS$&lt;서비스 이름&gt; 속성(고급 탭)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] **NS$***<user_configured name>* 형식으로 이름이 지정된 서비스로 구현됩니다. 사용자 지정 속성을 정의한 경우 속성이 해당 값과 함께 이 탭에 나타납니다. 이 탭에 속성이 표시되지 않을 수도 있습니다.
