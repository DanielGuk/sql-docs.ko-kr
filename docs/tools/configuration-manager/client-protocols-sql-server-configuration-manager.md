# <a name="client-protocols-sql-server-configuration-manager"></a>클라이언트 프로토콜(SQL Server 구성 관리자)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]프로토콜에서 사용 되는 순서를 구성 하려면는 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 이 컴퓨터의 클라이언트에서 마우스 오른쪽 단추로 클릭 **ClientProtocols** 클릭 한 다음 확인 하 고 콘솔 창에서 **속성**, 또는 마우스 오른쪽 단추로 클릭는 클릭 한 다음 확인 하 고 세부 정보 창에서 프로토콜 **순서**합니다.  
  
 이 컴퓨터에서 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 클라이언트가 사용하는 클라이언트 프로토콜의 속성을 구성하려면 콘솔 창에서 **ClientProtocols**를 클릭하고 세부 정보 창에서 원하는 프로토콜을 마우스 오른쪽 단추로 클릭한 다음 **속성**을 클릭합니다.
