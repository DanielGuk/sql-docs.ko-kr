# <a name="appliance-topology"></a>어플라이언스 토폴로지
전체 분석 플랫폼 시스템에 요약 정보를 표시합니다.  
  
전체 어플라이언스에 적용 되는 정보를 구성 하려면이 섹션의 요소를 확장 합니다. 병렬 데이터 웨어하우스 토폴로지 또는 HDInsight 토폴로지 섹션에 어플라이언스의 해당 지역에만 적용 되는 정보를 구성 하려면 선택 합니다.
