# <a name="appliance-time-zone-configuration---analytics-platform-system"></a>어플라이언스 표준 시간대 구성-분석 플랫폼 시스템
**시간대** 페이지를 사용 하면 Analytics Platform System (APS) 어플라이언스에 모든 노드에 대 한 표준 시간대를 설정할 수 있습니다.  
  
## <a name="to-set-the-time-zone"></a>표준 시간대를 설정 하려면  
  
1.  구성 관리자를 시작 합니다. 자세한 내용은 참조 [구성 관리자를 시작 &#40;분석 플랫폼 시스템&#41;](launch-the-configuration-manager.md)합니다.  
  
2.  어플라이언스 서비스를 사용 하 여 중지는 **서비스 상태** 구성 관리자의 페이지입니다. 참조 [PDW 서비스 상태 &#40;분석 플랫폼 시스템&#41; ](pdw-services-status.md) 지침에 대 한 합니다.  
  
3.  구성 관리자의 왼쪽된 창에서 클릭 **시간대**합니다. 원하는 시간대를 선택는 **시간대** 드롭 다운 메뉴. 현재 위치에 따라 선택할 수도 있습니다를 옆에 선택 **일광 절약 시간제에 맞게 시계를 자동으로 조정할**합니다.  
  
4.  클릭 **적용** 변경 내용을 저장 합니다.  
  
5.  어플라이언스 서비스를 사용 하 여 다시 시작의 **서비스 상태** 구성 관리자의 페이지입니다. 권한을 변경 하려면 또한 하려는 경우에 기기를 다시 시작 하기 전에 수행할 수 있습니다.
