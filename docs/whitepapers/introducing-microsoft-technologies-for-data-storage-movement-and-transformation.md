# 데이터 저장, 이동 및 변환을 위한 Microsoft 기술 소개
**요약:**이 문서는 데이터 저장, 이동 및 변환을 위한 Microsoft 기술을 소개하기 위한 것입니다. 이러한 기술은 조직이 분석 및 보고를 위해 유형이 다른 품질 데이터를 클라우드나 온-프레미스에서 저장 및 액세스할 수 있도록 하여 비즈니스 인텔리전스를 지원합니다.  
  
Microsoft 비즈니스 인텔리전스 기술에 대한 개요는 [Introducing Microsoft BI Reporting and Analysis Tools](https://msdn.microsoft.com/en-us/library/dn655131.aspx)(Microsoft BI보고 및 분석 도구 소개)를 참조하세요.
