# <a name="using-mathematical-functions"></a>수치 연산 함수 사용
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  수치 연산 함수는 숫자 식에 대해 수치 연산을 수행하고 연산 결과를 반환합니다.  
  
 기본적으로 수치 연산 함수로 전달되는 숫자는 배정밀도 부동 소수점 수로 해석됩니다.
