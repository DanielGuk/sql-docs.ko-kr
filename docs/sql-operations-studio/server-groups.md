# <a name="server-groups-in-includename-sosincludesname-sos-shortmd"></a>서버 그룹 [!INCLUDE[name-sos](../includes/name-sos-short.md)]

서버 그룹에는 서버 및 사용 하는 데이터베이스에 대 한 연결이 구성 하는 방법을 제공 합니다. 서버 그룹을 만들 때는 구성 세부 정보에 저장 되 고 *사용자 설정*합니다.

## <a name="create-and-edit-server-groups"></a>서버 그룹 만들기 및 편집

1. 클릭 **새 서버 그룹** 맨 위에 있는 *서버* 사이드바 합니다.
2. 그룹 이름을 입력 하 고 그룹에 대 한 색을 선택 합니다. 필요에 따라 설명을 추가 합니다.

   ![서버 그룹 추가](./media/server-groups/add-server-group.png)

기존 서버 그룹을 편집 하려면 그룹을 마우스 오른쪽 단추로 클릭 하 고 선택 **서버 그룹 편집**합니다.

사용 가능한 서버 그룹 색을 편집 하려면 편집할는 *서버 그룹* 값 [사용자 설정](settings.md)합니다.

> [!TIP]
> 끌어 놓은 서로 다른 서버 그룹 간의 서버 수 있습니다.
