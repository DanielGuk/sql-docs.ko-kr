# <a name="extend-the-functionality-of-includename-sosincludesname-sos-shortmd"></a>기능을 확장 [!INCLUDE[name-sos](../includes/name-sos-short.md)]

확장 [!INCLUDE[name-sos](../includes/name-sos-short.md)] 기준에 더 많은 기능을 추가 하면 쉽게 [!INCLUDE[name-sos](../includes/name-sos-short.md)] 설치 합니다. 

확장 (사용자)! 3rd 파티 커뮤니티 뿐만 아니라 SQL 작업 Studio 팀 (Microsoft)에서 제공 됩니다. 확장을 만드는 방법에 대 한 세부 정보를 참조 하십시오. [확장성 시작](https://github.com/Microsoft/sqlopsstudio/wiki/Getting-started-with-Extensibility)합니다.


## <a name="add-sql-operations-studio-extensions"></a>SQL 작업 Studio 확장 추가

1. 확장명 관리자를 열고 사용 가능한 확장에 액세스를 하려면 확장 아이콘을 선택 하거나 선택 **확장** 에 **보기** 메뉴.
2. 세부 정보를 보려면 사용 가능한 확장을 선택 합니다.

   ![확장 관리자](media/extensions/extension-manager.png)

1. 원하는 확장을 선택 하 고 **설치** 것입니다.
2. 선택 **다시 로드** (처음으로 확장을 설치 하면 필수)만 확장을 사용 하 합니다.
1. 서버 또는 데이터베이스를 마우스 오른쪽 단추로 클릭 하 고 선택 하 여 관리 대시보드를 탐색 **관리**합니다.
2. 설치 된 확장에 관리 대시보드 탭으로 표시 됩니다.

   ![확장 관리자](media/extensions/dashboard-extensions.png)
