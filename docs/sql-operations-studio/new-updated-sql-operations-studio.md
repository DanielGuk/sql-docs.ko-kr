# <a name="new-and-recently-updated-sql-operations-studio-docs"></a>새로 추가 되거나 최근에 업데이트: SQL Operations Studio docs

- 업데이트 날짜 범위: &nbsp; **2018-02-03**&nbsp; - &nbsp;**2018-04-28**
