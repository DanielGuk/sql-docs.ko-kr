# <a name="tabular-model-projects-ssas-tabular"></a>테이블 형식 모델 프로젝트(SSAS 테이블 형식)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]새 테이블 형식 모델을 만들려면 먼저 만들어야 합니다에서 새로운 테이블 형식 모델 프로젝트 [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]합니다. [!INCLUDE[ssBIDevStudio](../../includes/ssbidevstudio-md.md)]를 설치하면 테이블 형식 모델 프로젝트를 만들 수 있는 세 개의 새로운 비즈니스 인텔리전스 프로젝트 템플릿이 제공됩니다. 각 프로젝트 템플릿은 모델 제작을 시작할 수 있는 프레임워크를 제공합니다. 이 섹션의 항목에서는 프로젝트 템플릿을 사용하여 새로운 테이블 형식 모델 프로젝트를 만드는 방법에 대해 설명합니다.
