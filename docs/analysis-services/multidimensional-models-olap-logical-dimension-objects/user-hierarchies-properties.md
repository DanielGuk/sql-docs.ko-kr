# <a name="user-hierarchies---properties"></a>사용자 계층-속성
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  다음 표에서는 사용자 정의 계층의 속성을 설명합니다.  
  
|속성|Description|  
|--------------|-----------------|  
|AllMemberName|계층의 All 멤버에 대한 기본 언어의 캡션을 포함합니다.|  
|AllowDuplicateNames|계층에서 중복 이름이 허용되는지 여부를 결정합니다. 사용 가능한 값은 True와 False입니다. 기본값은 True입니다.|  
|Description|계층 설명을 포함합니다.|  
|DisplayFolder|사용자에 대한 계층을 나열할 폴더를 지정합니다.|  
|ID|계층의 고유 ID를 포함합니다.|  
|MemberNamesUnique|계층에서 멤버 이름이 고유해야 하는지 여부를 결정합니다. 사용 가능한 값은 True와 False입니다. 기본값은 False입니다.|  
|이름|계층 이름을 포함합니다.|
