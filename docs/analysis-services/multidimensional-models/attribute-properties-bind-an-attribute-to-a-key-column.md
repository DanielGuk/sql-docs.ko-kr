# <a name="attribute-properties---bind-an-attribute-to-a-key-column"></a>속성을 특성-키 열에 특성 바인딩
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]이 절차에 대 한 설정을 변경 하는 방법에 설명 된 **이름** 차원에서 특성의 속성입니다.  
  
### <a name="to-bind-an-attribute-to-a-key-column"></a>키 열에 특성을 바인딩하려면  
  
1.  차원 디자이너에서 이름을 바꿀 특성이 포함된 차원을 엽니다.  
  
2.  차원 작성기 보기의 **특성** 창에서 표시 형식을 **트리** 나 **목록**으로 변경합니다. 특성 창의 서식을 변경하는 방법은 [차원 디자이너에서 트리, 목록 또는 표 형태로 특성 보기](../../analysis-services/multidimensional-models/view-attributes-in-dimension-designer.md)를 참조하세요.  
  
3.  구성할 특성을 마우스 오른쪽 단추로 클릭한 다음 **이름 바꾸기**를 클릭합니다.  
  
4.  새 이름을 입력합니다.  
  
    > [!NOTE]  
    >  **의** 속성 **창에서 선택한 특성의** Name [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]속성을 설정할 수도 있습니다.
