# <a name="view-attributes-in-dimension-designer"></a>차원 디자이너에서 특성 보기
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]이 절차에서는 특성에 차원 디자이너에서에서 표시 되는 방식을 변경 하는 방법을 설명 합니다.  
  
### <a name="to-change-the-format-of-the-attributes-pane-in-dimension-designer"></a>차원 디자이너에서 특성 창의 형식을 변경하려면  
  
1.  차원 디자이너에서 작업할 특성이 포함된 차원을 열고 차원 구조 탭을 클릭합니다.  
  
2.  **특성** 창에서 아무 곳이나 클릭하여 **특성**창에 포커스를 둡니다.  
  
3.  **차원**메뉴를 클릭하거나 **특성** 창을 마우스 오른쪽 단추로 클릭합니다.  
  
4.  **특성 표시 위치**를 가리킨 다음 **트리**, **표**또는 **목록**을 클릭합니다.
