# SQL Server Data Tools에서 MDX 스크립트 만들기 및 실행
  [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)]에서 MDX 스크립트를 만들고 실행하려면 큐브를 미리 만들고 편집할 준비가 된 상태로 [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] 에 있어야 합니다.  
  
### MDX(Multidimensional Expressions) 스크립트를 만들려면  
  
1.  MDX 스크립트를 만들 큐브를 열고 **보기**에서 **계산**을 클릭합니다.  
  
2.  **스크립트 보기** 모드가 아닌 경우 **스크립트 보기** 아이콘을 클릭합니다.  
  
3.  스크립트 창의 CALCULATE 명령 아래에 MDX 식을 입력합니다. **구문 검사** 아이콘을 클릭하고 식이 구문상 올바른지 확인합니다.  
  
4.  MDX 스크립트를 실행하려면 새 MDX 스크립트 변경 내용이 포함된 큐브를 배포하고 처리합니다.
