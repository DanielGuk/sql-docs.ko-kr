# <a name="productlabel-element"></a>ProductLabel 요소
  제품 수준을 결정합니다.  
  
## <a name="syntax"></a>구문  
  
```xml  
  
<Server>  
...  
   <ProductLevel>...</ProductLevel>  
...  
</Server>  
```

|특징|설명|  
|--------------------|-----------------|  
|데이터 형식 및 길이|문자열|
