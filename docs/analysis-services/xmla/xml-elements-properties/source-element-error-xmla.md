# <a name="source-element-error-xmla"></a>Source 요소(Error)(XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]
  부모 [Error](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md) 요소를 생성한 구성 요소의 이름을 포함합니다.  
  
## <a name="syntax"></a>구문  
  
```xml  
  
<Error>  
   ...  
   <Source>...</Source>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>요소 특징  
  
|특징|설명|  
|--------------------|-----------------|  
|데이터 형식 및 길이|문자열|  
|기본값|없음|  
|카디널리티|0-1: 한 번만 나타날 수 있는 선택적 요소입니다.|  
  
## <a name="element-relationships"></a>요소 관계  
  
|관계|요소|  
|------------------|-------------|  
|부모 요소|[오류](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)|  
|자식 요소|없음|
