# <a name="command-events-event-category"></a>Command Events 이벤트 범주
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  Command Events 이벤트 범주에는 다음 표에 설명된 이벤트 클래스가 있습니다.  
  
|Event Class|이벤트 ID|Description|  
|-----------------|--------------|-----------------|  
|Command Begin|15|추적이 시작된 이후의 명령 시작 이벤트를 모두 수집합니다.|  
|Command End|16|추적이 시작된 이후의 명령 종료 이벤트를 모두 수집합니다.|  
  
 각각의 Command Events 이벤트 클래스와 연관된 열에 대한 자세한 내용은 [Command Events Data Columns](../../analysis-services/trace-events/command-events-data-columns.md)을 참조하십시오.
