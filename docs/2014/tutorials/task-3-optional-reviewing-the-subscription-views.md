# <a name="task-3-optional-reviewing-the-subscription-views"></a>태스크 3(선택 사항): 구독 뷰 검토
  이 작업에서는 SQL Server Management Studio를 사용하여 SQL 뷰가 만들어졌는지 확인합니다.  
  
1.  시작 **SQL Server Management Studio**합니다. 클릭는 **시작** 단추를 클릭 하 여 **모든 프로그램**, 클릭 **Microsoft SQL Server 2012**, 클릭 하 고 **SQL Server Management Studio**합니다.  
  
2.  에 **서버에 연결** 창의 설정 **서버 유형** 를 **데이터베이스 엔진**, 형식은 **서버 이름** (또는 선택 **( local)**, 적절 한 선택 **인증**를 클릭 하 고 **연결**합니다.  
  
3.  에 **개체 탐색기** 창 확장 **데이터베이스**, 확장 **MDS**, 펼친 다음 **뷰**합니다.  
  
4.  표시 되는지 확인은 **mdm. 공급 업체** 보기 목록에 있습니다.
