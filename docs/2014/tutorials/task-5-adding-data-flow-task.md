# <a name="task-5-adding-data-flow-task"></a>태스크 5: 데이터 흐름 태스크 추가
  이 작업에서는 SSIS 패키지의 제어 흐름에 데이터 흐름 태스크를 추가합니다.  
  
1.  끌어서 놓기 **데이터 흐름 태스크** 에서 **SSIS 도구 상자** 에 **제어 흐름** SSIS 디자이너의 탭 합니다. 표시 되지 않으면는 **SSIS 도구 상자**을 아무 곳 이나 클릭는 **제어 흐름** 탭을 클릭 **SSIS** 클릭 메뉴 모음에서 **SSIS 도구 상자**합니다.  
  
     ![끌어서 놓기 데이터 흐름 태스크가 SSIS 도구 상자에서](../../2014/tutorials/media/et-addingdataflowtask.jpg "끌어서 놓기 데이터 흐름 태스크가 SSIS 도구 상자에서")  
  
2.  마우스 오른쪽 단추로 클릭는 **데이터 흐름 태스크** 에 **제어 흐름** 탭을 클릭 **이름 바꾸기**합니다.  
  
3.  형식 **Receive, Cleanse, 일치 및 Curate Supplier Data** 누릅니다 **ENTER**합니다.  
  
4.  두 번 클릭은 **데이터 흐름 태스크** 전환할 수는 **데이터 흐름** 탭 합니다.
