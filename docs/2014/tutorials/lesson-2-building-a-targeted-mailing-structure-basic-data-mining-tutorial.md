# <a name="lesson-2-building-a-targeted-mailing-structure-basic-data-mining-tutorial"></a>2 단원: 타겟된 메일링 구조 (기본 데이터 마이닝 자습서)을 구축
  [!INCLUDE[ssSampleDBCoFull](../includes/sssampledbcofull-md.md)]의 마케팅 부서에서 특정 고객 대상의 메일 캠페인을 실시하여 판매를 증진하고자 합니다. 회사의 데이터베이스에는 기존 고객 목록과 새 잠재 고객 목록이 포함 되어 있습니다. 이전 고객의 특성을 조사 하 여 회사에서는 나중에 찾아낸 다음 잠재 고객에 적용할 수 있는 패턴을 검색 합니다. 자전거를 구매할 가능성이 가장 높은 삼을 잠재 고객을 예측할 수 과거 추세를 사용 예를 들어 [!INCLUDE[ssSampleDBCoFull](../includes/sssampledbcofull-md.md)], 하거나 이후 마케팅 캠페인에 대 한 고객 세그먼트를 만듭니다.  
  
 이 단원에서는 사용 하 여는 **데이터 마이닝 마법사** 타겟된 메일링 구조를 만들려고 합니다. 이 단원의 태스크를 마치면 단일 모델이 있는 마이닝 구조가 만들어집니다. 구조 만들기와 관련된 여러 단계와 중요한 개념이 있으므로 이 프로세스를 다음 3개의 태스크로 구분했습니다.
