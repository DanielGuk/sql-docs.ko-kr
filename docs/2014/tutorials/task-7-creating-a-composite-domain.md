# <a name="task-7-creating-a-composite-domain"></a>태스크 7: 복합 도메인 만들기
  이 태스크에서는 복합 도메인을 만들면 **Address Validation**로 구성 된 **Address Line**, **도시**, **상태**, 및  **Zip** 도메인입니다. 복합 도메인을 사용하면 규칙에 여러 도메인이 포함되는 도메인 간 규칙을 정의할 수 있습니다. 복합 도메인에는 필드 값을 여러 도메인으로 구문 분석할 수 있는 등의 다른 이점이 있습니다.  예를 들어 Full Name 필드에 대한 값을 각각 First Name, Middle Name 및 Last Name 도메인으로 구문 분석할 수 있습니다. 이 자습서에서는 도메인 간 규칙만 정의합니다. 참조 [복합 도메인 관리](http://msdn.microsoft.com/library/hh510399.aspx) 내용을 확인 합니다.  
  
1.  왼쪽된 창에서 클릭 **복합 도메인 만들기** 도구 모음 단추입니다.  
  
     ![복합 도메인 만들기 도구 모음 단추](../../2014/tutorials/media/et-creatingacompositedomain-01.jpg "는 복합 도메인 만들기 도구 모음 단추")  
  
2.  입력 **유효성 검사 주소** 에 대 한는 **복합 도메인 이름**합니다.  
  
     ![유효성 검사 복합 도메인 주소](../../2014/tutorials/media/et-creatingacompositedomain-02.jpg "유효성 검사 복합 도메인 주소")  
  
3.  도메인 목록에서 선택 **Address Line**, **도시**, **상태**, 및 **Zip** 클릭 **오른쪽 화살표** 에 추가 하는 **복합 도메인의 도메인** 목록입니다.  
  
4.  **확인** 을 클릭하여 대화 상자를 닫습니다.
