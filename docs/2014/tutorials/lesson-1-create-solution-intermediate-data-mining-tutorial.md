# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>1단원: 중간 데이터 마이닝 솔루션 만들기(중급 데이터 마이닝 자습서)
  기본 데이터 마이닝 자습서에서 만든는 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 간단한 데이터 마이닝 솔루션에 포함 된 프로젝트를 기반으로 새 [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)] 데이터베이스입니다.  
  
 이 자습서에 대 한 별도 만들어집니다 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 사용 하 여 프로젝트 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]합니다. 만듭니다는 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 사용 하는 데이터 원본 [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)], 시나리오를 지원 및 모델 유형을 하도록 해당 데이터 원본에 몇 가지 새 데이터 원본 뷰를 추가 합니다.  
  
 이 단원은 다음 태스크로 구성되어 있습니다.
