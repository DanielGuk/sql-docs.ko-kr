# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>1단원: 중간 데이터 마이닝 솔루션 만들기(중급 데이터 마이닝 자습서)
  기본 데이터 마이닝 자습서에서 만든를 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 새 단순 데이터 마이닝 솔루션을 포함 하는 프로젝트 기반 [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)] 데이터베이스입니다.  
  
 이 자습서에서는 별도 만들어집니다 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 프로젝트를 사용 하 여 [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]입니다. 만들려는 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 사용 하는 데이터 원본 [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)], 및 형식 모델 시나리오를 지원 하도록 해당 데이터 원본에 몇 가지 새 데이터 원본 뷰를 추가 합니다.
