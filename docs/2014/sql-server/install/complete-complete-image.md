# <a name="complete---complete-image"></a>완료 - 이미지 완료
  **완료** 페이지를 사용하여 준비 인스턴스 구성이 완료되었음을 확인할 수 있습니다.  
  
## <a name="options"></a>변수  
 이 대화 상자에는 이 작업에 대한 설치 로그 파일 링크가 포함됩니다. 설치를 완료하려면 **닫기**를 클릭합니다.
