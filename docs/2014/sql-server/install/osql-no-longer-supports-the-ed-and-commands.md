# <a name="osql-no-longer-supports-the-ed-and--commands"></a>osql은 더 이상 ED 및 !! 명령
  **osql** 유틸리티 지원 하지 않습니다는 **ED** 및 **!!** 명령을 지원하지 않습니다.  
  
## <a name="corrective-action"></a>수정 동작  
 에 대 한 참조 제거는 **ED** 및 **!!** 명령에 대한 참조를 제거합니다.  
  
 사용 하려는 경우는 **ED** 및 **!!** 명령을 사용 하 여는 **sqlcmd** 대신 유틸리티 **osql**합니다.
