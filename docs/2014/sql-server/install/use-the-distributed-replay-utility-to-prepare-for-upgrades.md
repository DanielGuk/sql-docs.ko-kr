# <a name="use-the-distributed-replay-utility-to-prepare-for-upgrades"></a>Distributed Replay Utility를 사용하여 업그레이드 준비
  [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Distributed Replay Utility 여러 컴퓨터에서 중요 한 작업 부하를 시뮬레이션에 추적 데이터를 재생 하려면 사용할 수 있습니다. [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 업그레이드 이전 및 이후에 테스트 서버에서 재생을 수행하면 성능 차이를 측정하고 업그레이드와의 응용 프로그램 비호환성을 확인할 수 있습니다. 자세한 내용은 [SQL Server Distributed Replay](../../tools/distributed-replay/sql-server-distributed-replay.md)을 참조하세요.
