# <a name="confirm-upgrade-advisor-settings"></a>업그레이드 관리자 설정 확인
  이 페이지에서는 선택한 업그레이드 관리자 분석 마법사 옵션과 입력한 매개 변수를 검토할 수 있습니다. 이러한 옵션과 매개 변수를 검토 한 후 클릭 **실행** 분석을 시작 하려면.  
  
## <a name="options"></a>변수  
 **업그레이드 관리자 설정**  
 각 구성 요소에 대해 입력한 모든 정보와 매개 변수뿐 아니라 모든 업그레이드 관리자에 적용되는 매개 변수를 검토할 수 있습니다. 여기에는 출력 디렉터리를 포함한 보고 옵션이 나열됩니다.  
  
 **[!INCLUDE[msCoName](../../includes/msconame-md.md)]에 보고서를 보냅니다. 이러한 보고서에는 개인 정보가 포함 될 수 있습니다.**  
 이 옵션을 선택 하면 분석을 전송 된 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 팀입니다. 이 정보는 이후 릴리스를 개선하는 데 사용됩니다.  
  
 **개인정보취급방침 봅니다.**  
 인터넷에 대한 [!INCLUDE[msCoName](../../includes/msconame-md.md)] 개인 정보 취급 방침을 읽으려면 이 링크를 클릭합니다.
