# <a name="behavior-changes-to-integration-services-features-in-sql-server-2014"></a>SQL Server 2014 Integration Services 기능의 동작 변경
  이 항목에서는 [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)]의 동작 변경 내용에 대해 설명합니다. 동작 변경 내용은 기능 작동 또는 현재 버전의 상호 작용 방식에 영향을 줄 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] 의 이전 버전을 비교 하 여 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]합니다.  
  
 Integration Services 기능의 동작은 SQL Server 2014에서 변경되지 않았습니다.
