# <a name="move-ssis-toolbox-items"></a>SSIS 도구 상자 항목 이동
  새 SSIS 도구 상자에는 기본 설정에 따라 항목을 구성할 수 있는 기능과 새 범주가 포함되어 있습니다. 도구 상자의 아래쪽에 있는 정보 창에는 선택한 도구 상자 항목에 대한 설명과 온라인 설명서 항목의 링크가 제공됩니다.  
  
### <a name="to-move-an-item-to-another-category"></a>항목을 다른 범주로 이동하려면  
  
1.  SSIS 도구 상자의 항목을 마우스 오른쪽 단추로 클릭하고 다음 중 하나를 클릭합니다.  
  
    -   **즐겨찾기로 이동**  
  
    -   **공용으로 이동**  
  
    -   **다른 원본으로 이동**  
  
    -   **다른 대상으로 이동**  
  
    -   **다른 변환으로 이동**  
  
    -   **다른 태스크로 이동**  
  
### <a name="to-refresh-the-ssis-toolbox"></a>SSIS 도구 상자를 새로 고치려면  
  
1.  SSIS 도구 상자를 마우스 오른쪽 단추로 클릭한 다음 **도구 상자 새로 고침**을 클릭합니다.
