# <a name="jobs-that-reference-a-schedule"></a>일정 참조 작업
  이 대화 상자를 사용하여 특정 일정을 참조하는 작업에 대한 정보를 확인할 수 있습니다.  
  
## <a name="options"></a>변수  
 **일정**  
 확인 중인 일정의 이름을 표시합니다.  
  
 **선택**  
 읽기 전용입니다.  
  
 **이름**  
 이 일정을 사용하는 작업의 이름입니다.  
  
 **Enabled**  
 읽기 전용입니다. 이 작업이 현재 사용하도록 설정되어 있는지 여부를 나타냅니다.  
  
 **범주**  
 작업 범주입니다.
