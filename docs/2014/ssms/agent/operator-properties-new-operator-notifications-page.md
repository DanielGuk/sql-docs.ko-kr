# <a name="operator-properties-new-operator-notifications-page"></a>연산자 속성: 새 운영자 (알림 페이지)
  이 페이지를 사용하여 운영자에게 알릴 경고 및 작업을 설정할 수 있습니다.  
  
## <a name="options"></a>변수  
 **경고**  
 인스턴스의 경고를 표시합니다.  
  
 **작업**  
 인스턴스의 작업을 표시합니다.  
  
 **경고 목록**  
 인스턴스의 경고를 나열합니다.  
  
 **작업 목록**  
 인스턴스의 작업을 나열합니다.  
  
 경고 목록과 작업 목록에서는 공통적으로 다음 옵션을 사용할 수 있습니다.  
  
 **전자 메일**  
 전자 메일을 사용하여 이 운영자에게 알립니다.  
  
 **호출기**  
 호출기 주소로 전자 메일을 보내 이 운영자에게 알립니다.  
  
 **Net send**  
 **net send**를 사용하여 이 운영자에게 알립니다.
