# <a name="collection-permissions-master-data-services"></a>컬렉션 권한(Master Data Services)
  컬렉션 권한은 엔터티의 모든 컬렉션에 적용됩니다. 특정 컬렉션에 사용 권한을 부여할 수는 없습니다. 사용 권한은 모든 컬렉션에 적용됩니다.  
  
> [!NOTE]  
>  이러한 사용 권한은 사용자 인터페이스의 **탐색기** 기능 영역에만 적용됩니다.  
  
|사용 권한|Description|  
|----------------|-----------------|  
|**읽기 전용**|모든 컬렉션 및 컬렉션 멤버가 표시됩니다.<br /><br /> 사용자는 컬렉션을 추가하거나 제거할 수 없으며 컬렉션에 멤버를 추가하거나 컬렉션에서 멤버를 제거할 수도 없습니다.|  
|**Update**|모든 컬렉션이 표시되고 사용자는 컬렉션을 추가 및 제거할 수 있습니다.<br /><br /> 또한 사용자는 컬렉션에서 멤버를 추가하거나 제거할 수도 있습니다.|  
|**거부**|엔터티의 일부 컬렉션이 표시되지 않습니다.|
