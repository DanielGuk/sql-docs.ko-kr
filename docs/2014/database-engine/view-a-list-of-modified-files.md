# <a name="view-a-list-of-modified-files"></a>수정된 파일의 목록 보기
  사용할 수는 **보류 중인 체크 인** 창에 표시 하려면 현재 솔루션의 체크 아웃 된 파일의 목록을 언제 및 체크 인 한 번의 단추를 사용 하 여 이러한 파일을 클릭 합니다.  
  
### <a name="to-view-a-list-of-modified-files"></a>수정된 파일의 목록을 보려면  
  
1.  에 **보기** 메뉴를 클릭 하 여 **보류 중인 체크 인**합니다.  
  
2.  선택한 파일을 확인 하려면 클릭 **체크 인**합니다. 고정할 수 있습니다는 **보류 중인 체크 인** 의 오른쪽 창에서는 [!INCLUDE[ssManStudio](../includes/ssmanstudio-md.md)] 환경 했으면 파일에서 확인할 수 있도록 작업 합니다.  
  
     **체크인**  
     솔루션을 체크 인합니다.  
  
     **설명**  
     일반 텍스트 설명을 보류 중인 체크 인에 연결합니다. 만들어진 설명은 각 버전의 프로젝트와 연결되며 원본 제어 데이터베이스에 저장됩니다.  
  
     **옵션**  
     소스 제어를 클릭할 때 수행 해야 하는 작업을 지정 된 **체크 인** 단추입니다.  
  
    -   **모든 체크 아웃 유지**  
  
         변경 내용이 원본 제어 공급자에 기록되지만 파일의 체크 아웃 상태는 유지하도록 지정합니다.  
  
     **Sort**  
     항목 목록에 표시되는 열의 정렬 순서를 지정합니다.  
  
     **열**  
     창의 항목 목록에 추가할 수 있는 열 목록을 표시합니다.  
  
     **트리 뷰**  
     체크 인하고 있는 솔루션이나 프로젝트의 폴더와 파일 계층 구조를 표시합니다.  
  
     **플랫 보기**  
     체크 인하고 있는 파일을 해당 원본 제어 연결 아래에 기본 목록으로 표시합니다.  
  
     **버전 비교**  
     Visual SourceSafe 엽니다 **차이점 옵션** 대화 상자를 선택한 다른 파일을 개발 환경 프로젝트에서 선택한 파일을 비교 하 고 있는 경우 차이 표시 합니다.  
  
     **체크 아웃 취소**  
     선택한 모든 항목의 체크 아웃을 취소는 **보류 중인 체크 인** 창.  
  
     **이름**  
     체크 인에 사용할 수 있는 항목 목록을 표시합니다. 선택한 항목 옆에는 확인란이 함께 나타납니다. 특정 항목을 체크 인하지 않으려면 옆에 있는 확인란의 선택을 취소합니다.
