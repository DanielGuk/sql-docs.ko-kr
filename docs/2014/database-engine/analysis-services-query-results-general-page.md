# <a name="analysis-services-query-results-general-page"></a>Analysis Services 쿼리 결과(일반 페이지)
  이 페이지를 사용하여 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 쿼리를 실행하는 옵션을 지정할 수 있습니다.  
  
 **실행 제한 시간**  
 쿼리를 취소할 때까지 기다리는 시간(초)을 나타냅니다.  
  
 값 0은 무한 대기 또는 제한 시간 없음을 의미합니다.  
  
 **기본값으로 다시 설정**  
 이 페이지의 모든 값을 원래 기본값으로 다시 설정합니다.
