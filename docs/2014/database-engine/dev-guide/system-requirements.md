# <a name="system-requirements"></a>시스템 요구 사항
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Management Objects (SMO)를 사용 하 여 [!INCLUDE[msCoName](../../includes/msconame-md.md)] `System.Data.SqlClient` 개체 드라이버를 연결 하 고 인스턴스와 통신 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]합니다.  
  
 SMO 클라이언트에 필요 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 에 포함 된 네이티브 클라이언트 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 고 [!INCLUDE[dnprdnshort](../../includes/dnprdnshort-md.md)] 2.0.  
  
 SMO를 사용 하 여 응용 프로그램을 개발 하려면 있어야 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 2008이 설치 되어 있습니다.
