# <a name="options-query-results-analysis-server-general-page"></a>옵션 (쿼리 결과 분석 서버-일반 페이지)
  이 페이지를 사용하여 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 쿼리에 대한 쿼리 결과 창의 기본 동작을 결정할 수 있습니다. 이 설정을 표시하려면 **도구** 메뉴에서 **옵션** 을 선택합니다. **쿼리 결과** 폴더에서 **분석 서버** 하위 폴더를 확장한 다음 **일반**을 선택합니다.  
  
## <a name="uielement-list"></a>UIElement 목록  
 **별도 탭에 표 형태 표시**  
 여러 개의 결과 집합이 별도의 문서로 표시되며 결과 창에서 순서대로 액세스하는 대신 문서 창의 탭을 사용하여 액세스할 수 있습니다.  
  
 **쿼리 일괄 처리 완료 시 Windows 기본 경고음 울림**  
 쿼리 일괄 처리가 완료되면 신호를 제공하도록 지정합니다.  
  
 **기본값으로 다시 설정**  
 이 페이지의 모든 값을 원래 기본값으로 다시 설정합니다.
