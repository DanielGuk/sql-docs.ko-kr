# <a name="reportparameters-element-assl"></a>ReportParameters 요소(ASSL)
  컬렉션을 포함 [ReportParameter](../objects/reportparameter-element-assl.md) 에 대 한 요소를 [ReportAction](../data-type/action-data-type-assl.md) 요소입니다.  
  
## <a name="syntax"></a>구문  
  
```xml  
  
<Action xsi:type="ReportAction">  
   ...  
   <ReportParameters>  
      <ReportParameter>...</ReportParameter>  
   </ReportParameters>  
   ...  
</Action>  
```  
  
## <a name="element-characteristics"></a>요소 특징  
  
|특징|Description|  
|--------------------|-----------------|  
|데이터 형식 및 길이|없음|  
|기본값|없음|  
|카디널리티|0-1: 한 번만 나타날 수 있는 선택적 요소입니다.|  
  
## <a name="element-relationships"></a>요소 관계  
  
|관계|요소|  
|------------------|-------------|  
|부모 요소|[ReportAction](../objects/action-element-assl.md) 유형의 [Action](../data-type/action-data-type-assl.md)|  
|자식 요소|[ReportParameter](../objects/reportparameter-element-assl.md)|  
  
## <a name="remarks"></a>Remarks  
 Analysis Management Objects (AMO) 개체 모델의 해당 요소는 <xref:Microsoft.AnalysisServices.ReportParameterCollection>합니다.
