# <a name="session-events-event-category"></a>Session Events 이벤트 범주
  Session Events 이벤트 범주에는 다음 표에 설명된 이벤트 클래스가 있습니다.  
  
|Event Class|이벤트 ID|Description|  
|-----------------|--------------|-----------------|  
|Existing Connection|41|추적이 시작된 이후의 연결 이벤트를 모두 수집합니다.|  
|Existing Session|42|추적이 시작된 이후의 세션 이벤트를 모두 수집합니다.|  
|Session Initialize|43|세션 초기화 이벤트를 수집합니다.|  
  
 각각의 Session Events 이벤트 클래스와 연관된 열에 대한 자세한 내용은 [Session Events Data Columns](session-events-data-columns.md)를 참조하십시오.
