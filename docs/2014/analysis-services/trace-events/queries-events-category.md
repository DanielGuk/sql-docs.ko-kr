# <a name="queries-events-category"></a>Queries Events 범주
  Queries Events 이벤트 범주에는 다음 표에 설명된 이벤트 클래스가 있습니다.  
  
|Event Class|이벤트 ID|Description|  
|-----------------|--------------|-----------------|  
|Query Begin|9|추적이 시작된 이후의 쿼리 시작 이벤트를 모두 수집합니다.|  
|쿼리 끝|10|추적이 시작된 이후의 쿼리 종료 이벤트를 모두 수집합니다.|  
  
 각각의 Query Events 이벤트 클래스와 연관된 열에 대한 자세한 내용은 [Queries Events Data Columns](queries-events-data-columns.md)을 참조하십시오.
