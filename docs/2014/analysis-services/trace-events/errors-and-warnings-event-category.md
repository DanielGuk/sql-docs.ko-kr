  Errors and Warnings 이벤트 범주에는 다음 표에 설명된 이벤트 클래스가 있습니다.  
  
|Event Class|이벤트 ID|Description|  
|-----------------|--------------|-----------------|  
|Error|17|추적이 시작된 이후의 새 오류 이벤트를 모두 기록합니다.|  
  
 각 Errors and Warnings 이벤트 클래스와 연관된 열에 대한 자세한 내용은 [Errors and Warnings Events Data Columns](errors-and-warnings-events-data-columns.md)를 참조하십시오.
