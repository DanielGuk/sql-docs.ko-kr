# <a name="completing-the-wizard-linked-object-wizard"></a>마법사 완료(연결된 개체 마법사)
  **마법사 완료** 페이지를 사용하여 마법사의 이전 페이지에서 결정한 사항을 검토할 수 있습니다.  
  
## <a name="options"></a>변수  
 **미리 보기**  
 연결 또는 가져오기를 위해 선택한 개체를 표시합니다.
