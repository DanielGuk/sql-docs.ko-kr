# <a name="completing-the-wizard-cube-wizard"></a>마법사 완료(큐브 마법사)
  **마법사 완료** 페이지를 사용하여 마법사가 완료되면 생성될 큐브의 이름을 지정하고 큐브를 미리 볼 수 있습니다.  
  
## <a name="options"></a>변수  
 **큐브 이름**  
 큐브의 이름을 입력합니다.  
  
 **미리 보기**  
 마법사가 완료되면 생성될 큐브의 구조를 표시합니다.
