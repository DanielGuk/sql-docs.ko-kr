# <a name="suggest-related-columns-data-mining-wizard"></a>관련 열 제안(데이터 마이닝 마법사)
  **관련 열 제안** 페이지를 사용하여 마법사의 이전 페이지에서 선택한 **제안** 기능의 권장 사항을 볼 수 있습니다. 입력 열은 선택한 **예측 가능** 열과의 연관성에 따라 미리 선택됩니다. 이 대화 상자의 변경 내용을 적용하는 경우 해당 변경 내용은 **학습 데이터 지정** 페이지에 반영됩니다.  
  
 **자세한 내용:** [데이터 마이닝 마법사&#40;Analysis Services - 데이터 마이닝&#41;](data-mining/data-mining-wizard-analysis-services-data-mining.md), [관계형 마이닝 구조 만들기](data-mining/create-a-relational-mining-structure.md)  
  
## <a name="options"></a>변수  
 **열 이름**  
 마법사의 **학습 데이터 지정** 페이지에서 정의한 입력 열을 표시합니다.  
  
 **점수**  
 해당 열과 이전에 선택한 **예측 가능** 열 사이의 관계를 표시합니다.  
  
 **Input**  
 예측 계산에 이 열을 포함하려면 선택합니다.
