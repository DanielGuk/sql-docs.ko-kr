# <a name="cube-selection-dialog-box"></a>큐브 선택 대화 상자
  **큐브 선택** 대화 상자를 사용하여 보고서의 데이터 원본으로 다른 큐브를 선택할 수 있습니다. 다른 큐브를 선택하면 사용자 쿼리는 더 이상 유효하지 않을 수 있습니다.
