# <a name="error-page-report-manager"></a>오류 페이지(보고서 관리자)
  오류 페이지를 사용하여 오류 상태에 대한 정보를 볼 수 있습니다. 서버 기반 또는 세션 기반 오류가 이 페이지에 표시됩니다. 특정 페이지 컨트롤과 관련된 유효성 검사 오류는 컨트롤 옆에 인라인으로 표시됩니다.  
  
-   로컬 보고서 서버를 탐색 하는 경우 다음과 유사한 오류가 표시 참조: [로컬 관리를 위한 기본 모드 보고서 서버 구성 &#40;SSRS&#41;](report-server/configure-a-native-mode-report-server-for-local-administration-ssrs.md)  
  
     사용자 ' 도메인\\[사용자 이름]' 권한이 필요 합니다. 충분한 권한이 부여되고 Windows UAC(사용자 계정 컨트롤) 제한이 해결되었는지 확인하세요.  
  
-   다음과 비슷한 오류 메시지가 표시 되 면 참조 [원격 관리를 위한 보고서 서버 구성](report-server/configure-a-report-server-for-remote-administration.md)합니다.  
  
     해당 컴퓨터를 찾을 수 없습니다. "RPC 서버를 사용할 수 없습니다. (HRESULT: 0x800706BA에서 예외가 발생했습니다.)"  
  
-   원격 서버에서 발생되는 오류 조건에 대한 추가 정보를 반환하도록 [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] 보고서 서버에 대한 서버 속성을 설정할 수 있습니다. 오류 메시지에 텍스트가 포함 된 "이이 오류에 대 한 자세한 내용은 로컬 서버 컴퓨터의 보고서 서버를 탐색 하거나 원격 오류 사용"을 참조 하세요 [원격 오류 사용 &#40;Reporting Services&#41;](report-server/enable-remote-errors-reporting-services.md)합니다.
