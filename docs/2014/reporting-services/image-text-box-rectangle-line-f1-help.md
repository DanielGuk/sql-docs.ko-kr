# <a name="image-text-box-rectangle-line-f1-help"></a>이미지, 입력란, 사각형, 선 F1 도움말
  이 섹션의 항목에서는 보고서 디자이너의 대화 상자에서 이미지, 입력란, 사각형 및 선에 설정할 수 있는 옵션에 대해 설명합니다.  
  
 이 항목에 액세스하려면 **보고서 속성** 대화 상자에서 **도움말**을 클릭합니다.
