# <a name="localdberrorwaittimeout"></a>LOCALDB_ERROR_WAIT_TIMEOUT
    
## <a name="details"></a>설명  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|277|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|로컬 데이터베이스 인스턴스 API 메서드 내에서 시간 초과가 발생했습니다.|  
  
## <a name="explanation"></a>설명  
 동기화 잠금을 획득하려고 시도하는 중에 시간 초과가 발생했습니다.  
  
## <a name="user-action"></a>사용자 동작  
 요청한 작업을 다시 시도하십시오.
