# <a name="localdberrorunknownversion"></a>LOCALDB_ERROR_UNKNOWN_VERSION
    
## <a name="details"></a>설명  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|269|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|지정된 LocalDB 버전을 이 컴퓨터에서 사용할 수 없습니다.|  
  
## <a name="explanation"></a>설명  
 지정한 버전을 사용할 수 없습니다.  
  
## <a name="user-action"></a>사용자 동작  
 요청한 로컬 데이터베이스 런타임 버전을 설치하십시오.
