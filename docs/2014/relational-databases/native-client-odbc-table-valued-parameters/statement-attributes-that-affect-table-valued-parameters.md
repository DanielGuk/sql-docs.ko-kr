# <a name="statement-attributes-that-affect-table-valued-parameters"></a>테이블 반환 매개 변수에 영향을 주는 문 특성
  다음 표에서는 설명자 필드의 특성을 설명합니다.  
  
|특성 이름|형식|Description|  
|--------------------|----------|-----------------|  
|SQL_SOPT_SS_PARAM_FOCUS|SQLUINTEGER|SQL_SS_PARAM_FOCUS에 대 한 자세한 내용은 참조 [SQLSetStmtAttr](../native-client-odbc-api/sqlsetstmtattr.md)합니다.|  
|SQL_SOPT_SS_NAME_SCOPE|SQLUINTEGER|SQL_SS_NAME_SCOPE에 대 한 자세한 내용은 참조 [SQLSetStmtAttr](../native-client-odbc-api/sqlsetstmtattr.md)합니다.|
