# <a name="keep-the-affinity-mask-default-value"></a>선호도 마스크 기본값 유지
  이 규칙은 선호도 마스크에 대한 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 설정이 기본값인 0으로 설정되어 있는지 검사합니다. 선호도 마스크 옵션은 CPU 선호도를 동적으로 제어합니다.
