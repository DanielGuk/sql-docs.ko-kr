# <a name="storage-system-input-output-time-out"></a>저장소 시스템 입/출력 시간 초과
  이 규칙은 시스템 이벤트 로그에서 EventId 9를 검사합니다. 이 메시지는 저장소 시스템에서 I/O 시간 초과가 발생했음을 나타냅니다.
