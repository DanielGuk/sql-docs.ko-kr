# <a name="sqlxmlparameter-object-sqlxml-managed-classes"></a>SqlXmlParameter 개체(SQLXML 관리되는 클래스)
  SqlXmlParameter 개체는 이러한 속성을 지원합니다.  
  
 속성  
 매개 변수의 이름입니다. 명령에 매개 변수가 전달될 수 있습니다. SqlXmlCommand 개체의 CreateParameter 메서드 호출 매개 변수 개체를 생성 합니다.  
  
 값  
 매개 변수의 값입니다. 작업 예제를 참조 하십시오. [SQL 쿼리 실행 &#40;SQLXML Managed Classes&#41;](sqlxml-4-0-net-framework-support-managed-classes.md)합니다.
