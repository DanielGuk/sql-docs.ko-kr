# <a name="servicename-property-sqlserviceadvancedproperty-class"></a>ServiceName 속성(SqlServiceAdvancedProperty 클래스)
  고급 속성과 연결된 관리되는 서비스의 이름을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.ServiceName [= value]  
```  
  
## <a name="parts"></a>부분  
 *object*  
 고급 속성을 나타내는 [SqlServiceAdvancedProperty 클래스](sqlserviceadvancedproperty-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 고급 속성과 연결된 서비스의 이름을 지정하는 문자열 값입니다.
