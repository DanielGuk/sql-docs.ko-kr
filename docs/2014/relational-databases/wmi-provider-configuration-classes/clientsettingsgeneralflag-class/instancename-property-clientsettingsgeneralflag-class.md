# <a name="instancename-property-clientsettingsgeneralflag-class"></a>InstanceName 속성(ClientSettingsGeneralFlag 클래스)
   [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 클라이언트 인스턴스의 이름을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>부분  
 *object*  
 서버 설정에 대한 일반 플래그를 나타내는 [ClientSettingsGeneralFlag 클래스](clientsettingsgeneralflag-class.md) 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 클라이언트 인스턴스의 이름을 지정하는 문자열 값입니다.
