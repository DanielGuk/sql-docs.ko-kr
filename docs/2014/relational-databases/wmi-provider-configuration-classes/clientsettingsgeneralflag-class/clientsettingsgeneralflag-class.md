# <a name="clientsettingsgeneralflag-class"></a>ClientSettingsGeneralFlag 클래스

 [ClientSettingsGeneralFlag 클래스](clientsettingsgeneralflag-class.md) 는 다음을 수행 하는 데 사용할 수 있습니다.

-   플래그 값을 보고 수정합니다.
