# <a name="sqlclosecursor"></a>SQLCloseCursor
  **SQLCloseCursor** 대체 [SQLFreeStmt](sqlfreestmt.md) 와 *옵션* SQL_CLOSE의 값입니다. 수신 시 **SQLCloseCursor**, [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client ODBC 드라이버는 보류 중인 결과 집합 행을 삭제 합니다. 문의 열 및 매개 변수 바인딩(있는 경우)은 **SQLCloseCursor**에 의해 변경되지 않습니다.
