# <a name="proxy-account-properties-references-tab"></a>프록시 계정 속성(참조 탭)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] 이 읽기 전용 페이지에는 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 에이전트 프록시 계정을 참조하는 작업 단계가 나열됩니다.  
  
## <a name="options"></a>변수  
**작업 이름**  
이 프록시 계정을 사용하는 작업의 이름입니다.  
  
**작업 단계**  
이 프록시 계정을 사용하는 작업에 있는 작업 단계의 이름입니다.  
  
**하위 시스템**  
이 프록시 계정에서 사용하는 하위 시스템입니다.
