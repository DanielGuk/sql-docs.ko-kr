# <a name="object-already-exists"></a>이미 존재하는 개체
저장하려는 개체의 이름이 데이터베이스에 이미 있는 개체의 이름과 같습니다. 다른 이름을 선택한 다음 계속하십시오.  
  
이미 있는 열 이름도 나타낼 수 있습니다.
