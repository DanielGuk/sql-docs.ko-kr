# <a name="view-join-properties-visual-database-tools"></a>조인 속성 보기(Visual Database Tools)
속성 창에서 뷰의 속성 컬렉션을 볼 수 있습니다.  
  
### <a name="to-view-join-properties"></a>조인 속성을 보려면  
  
1.  **다이어그램** 창에서 조인을 마우스 오른쪽 단추로 클릭합니다.  
  
2.  바로 가기 메뉴에서 **속성**을 클릭합니다.  
  
    속성 창에 조인의 속성이 표시됩니다.
