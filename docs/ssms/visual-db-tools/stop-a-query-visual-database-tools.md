# <a name="stop-a-query-visual-database-tools"></a>쿼리 중지(Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
쿼리에 시간이 너무 오래 걸리거나 예상한 결과를 반환하지 않은 경우 아직 완료되지 않았으면 쿼리를 중지할 수 있습니다.  
  
### <a name="to-stop-a-query"></a>쿼리를 중지하려면  
  
-   [결과 창](../../ssms/visual-db-tools/results-pane-visual-database-tools.md)에서 임의의 위치를 마우스 오른쪽 단추로 클릭하고 바로 가기 메뉴에서 **창**을 가리킨 다음 **결과 지우기**를 클릭합니다.
