# <a name="change-the-default-location-for-projects"></a>프로젝트 기본 위치 변경
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
파일 및 프로젝트의 기본 위치는 내 문서 폴더의 [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 프로젝트 폴더입니다.  
  
### <a name="to-change-the-default-location-for-projects"></a>프로젝트의 기본 위치를 변경하려면  
  
1.  다른 위치에서 파일 또는 프로젝트를 열거나 저장하여 기본 위치를 변경합니다. [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 이전에 열린 프로젝트의 위치를 저장하였다가 다음에 프로젝트를 열거나 저장할 때 해당 위치를 사용합니다.
