# <a name="browse-for-servers-local-servers"></a>서버 찾아보기(로컬 서버)
[!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 구성 요소에 연결할 때 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]의 정확한 이름을 모르는 경우 **서버 이름** 상자에서 **더 찾아보기**를 클릭하여 **서버 찾아보기** 대화 상자를 엽니다.  
  
## <a name="options"></a>옵션  
**연결할 서버를 선택하십시오.**  
트리에 표시된 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 인스턴스를 클릭하여 연결할 서버를 지정합니다. **+** 또는 **–** 기호로 표시된 노드를 클릭하여 트리 뷰의 일부분을 표시하거나 숨길 수 있습니다.
