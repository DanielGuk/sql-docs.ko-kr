# <a name="password-expired"></a>암호 만료
[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 인증을 사용하여 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull_md.md)]가 있는 서버에 연결할 때 제공한 암호가 올바르지만 만료된 경우에 나타납니다. **다음 로그인할 때 반드시 암호 변경** 옵션을 사용하여 만든 새 계정으로 서버에 연결할 때도 나타납니다. **암호 만료** 대화 상자를 사용하여 이 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 인증 로그인의 암호를 변경할 수 있습니다.  
  
## <a name="options"></a>옵션  
**로그인**  
사용 중인 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 로그인을 표시합니다.  
  
**새 암호**  
새 로그인 암호를 입력합니다.  
  
**암호 확인**  
새 암호를 다시 입력하여 제대로 입력했는지 확인합니다.
