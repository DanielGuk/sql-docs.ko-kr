# <a name="lesson-2-3---maximizing-query-editor"></a>2-3단원 - 쿼리 편집기 화면 크기
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 프로그래머가 자주 제기하는 "코드를 쓸 공간을 더 확보하려면 어떻게 합니까?"라는 질문에 대해 두 가지 간단한 방법이 있다고 대답할 수 있습니다. 쿼리 편집기 창을 최대화하고 사용하지 않는 도구 창을 숨기면 됩니다.  
  
## <a name="maximizing-the-query-editor-window"></a>쿼리 편집기 창 최대화  
  
#### <a name="to-maximize-the-query-editor-window"></a>쿼리 편집기 창을 최대화하려면  
  
1.  쿼리 편집기 창의 아무 곳이나 클릭합니다.  
  
2.  Shift+Alt+Enter를 눌러 전체 화면 모드와 일반 모드 사이를 전환합니다.  
  
    이 바로 가기 키는 모든 문서 창에서 작동합니다.  
  
## <a name="hiding-tool-windows"></a>도구 창 숨기기  
  
#### <a name="to-automatically-hide-all-tool-windows"></a>모든 도구 창을 자동으로 숨기려면  
  
1.  쿼리 편집기 창의 아무 곳이나 클릭합니다.  
  
2.  **창** 메뉴에서 **모두 자동 숨기기**를 클릭합니다.  
  
3.  도구 창을 복원하려면 각 도구를 연 다음 창에서 **자동 숨기기** 단추를 클릭하여 열린 상태로 고정합니다.
