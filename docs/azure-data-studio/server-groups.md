# <a name="server-groups-in-includename-sosincludesname-sos-shortmd"></a>서버 그룹 [!INCLUDE[name-sos](../includes/name-sos-short.md)]

서버 그룹에 서버를 사용 하 여 작업할 데이터베이스 연결을 구성 하는 방법을 제공 합니다. 구성 세부 정보에 저장 된 서버 그룹을 만들 때 *사용자 설정*합니다.

## <a name="create-and-edit-server-groups"></a>만들기 및 서버 그룹 편집

1. 클릭 **새 서버 그룹** 맨 위에 있는 합니다 *서버* 보충 합니다.
2. 그룹 이름을 입력 하 고 그룹에 대 한 색을 선택 합니다. 선택적으로 설명을 추가 합니다.

   ![서버 그룹 추가](./media/server-groups/add-server-group.png)

기존 서버 그룹을 편집 하려면 그룹을 마우스 오른쪽 단추로 클릭 하 고 선택 **서버 그룹 편집**합니다.

사용 가능한 서버 그룹 색을 편집 하려면 편집 합니다 *서버 그룹* 에 값 [사용자 설정](settings.md)합니다.

> [!TIP]
> 끌어 하 고 다른 서버 그룹 사이 서버를 삭제할 수 있습니다.
