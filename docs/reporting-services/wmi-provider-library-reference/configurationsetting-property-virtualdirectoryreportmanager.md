# <a name="configurationsetting-property---virtualdirectoryreportmanager"></a>ConfigurationSetting 속성 - VirtualDirectoryReportManager

## <a name="syntax"></a>구문  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## <a name="property-values"></a>속성 값  
 가상 디렉터리가 들어 있는 **string** 입니다.
