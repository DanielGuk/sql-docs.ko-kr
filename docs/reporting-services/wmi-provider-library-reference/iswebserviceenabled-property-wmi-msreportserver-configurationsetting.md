# IsWebServiceEnabled 속성(WMI MSReportServer_ConfigurationSetting)
  보고서 서버 웹 서비스를 사용하는지 여부를 나타냅니다. 읽기 전용입니다.  
  
## 구문  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## 속성 값  
 읽기 전용 **부울** 값입니다. **true** 값은 보고서 서버 웹 서비스를 사용할 수 있음을 나타냅니다.
