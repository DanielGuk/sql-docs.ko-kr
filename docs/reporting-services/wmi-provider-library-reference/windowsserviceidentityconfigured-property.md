# WindowsServiceIdentityConfigured 속성(WMI MSReportServer_ConfigurationSetting)
  보고서 서버 Windows 서비스를 실행하도록 마지막으로 구성된 ID를 반환합니다. 읽기 전용입니다.  
  
## 구문  
  
```vb  
Public Dim WindowsServiceIdentityConfigured As String  
```  
  
```csharp  
public string WindowsServiceIdentityConfigured;  
```  
  
## 속성 값  
 보고서 서버 Windows 서비스를 실행하도록 마지막으로 구성된 ID가 들어 있는 **String** 값입니다.
