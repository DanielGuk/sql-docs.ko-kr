# InstallationID 속성(WMI MSReportServer_ConfigurationSetting)
  이 속성은 특정 보고서 서버 인스턴스의 고유 식별자를 반환합니다.  
  
## 구문  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
