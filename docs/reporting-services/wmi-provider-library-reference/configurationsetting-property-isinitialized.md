# <a name="configurationsetting-property---isinitialized"></a>IsInitialized ConfigurationSetting 속성-
  보고서 서버 인스턴스가 초기화되었는지 여부를 나타냅니다. 읽기 전용입니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
Public Dim IsInitialized As Boolean  
```  
  
```csharp  
public Boolean DatabaseLogonName;  
```  
  
## <a name="property-values"></a>속성 값  
 보고서 서버 인스턴스가 초기화되었는지 여부를 나타내는 **Boolean** 개체입니다.
