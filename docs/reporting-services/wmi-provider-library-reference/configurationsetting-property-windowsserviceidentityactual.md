# <a name="configurationsetting-property---windowsserviceidentityactual"></a>WindowsServiceIdentityActual ConfigurationSetting 속성-
  보고서 서버의 Windows 서비스가 실행되고 있는 ID입니다. 읽기 전용입니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
Public Dim WindowsServiceIdentityActual As String  
```  
  
```csharp  
public string WindowsServiceIdentityActual;  
```  
  
## <a name="property-values"></a>속성 값  
 보고서 서버의 Windows 서비스가 실행되고 있는 ID가 들어 있는 **String** 개체입니다.
