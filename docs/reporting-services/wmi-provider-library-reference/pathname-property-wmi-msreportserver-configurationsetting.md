# PathName 속성(WMI MSReportServer_ConfigurationSetting)
  보고서 서버 인스턴스의 설치 경로를 지정합니다. 읽기 전용입니다.  
  
## 구문  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## 속성 값  
 설치 경로를 나타내는 **String** 개체입니다.
