# MSReportServer_Instance 속성
  MSReportServer_Instance 클래스는 클라이언트에서 설치된 보고서 서버에 연결하는 데 필요한 기본 정보를 제공합니다.  
  
## Public 속성  
  
|||  
|-|-|  
|[EditionID 속성&#40;WMI MSReportServer_Instance&#41;](../../reporting-services/wmi-provider-library-reference/editionid-property-wmi-msreportserver-instance.md)|보고서 서버의 버전을 반환합니다.|  
|[EditionName 속성&#40;WMI MSReportServer_Instance&#41;](../../reporting-services/wmi-provider-library-reference/editionname-property-wmi-msreportserver-instance.md)|사람이 읽을 수 있는 보고서 서버 버전의 이름을 반환합니다.|  
|[InstanceID](../../reporting-services/wmi-provider-library-reference/instanceid-property-wmi-msreportserver-instance.md)|이 개체는 [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.|  
|[InstanceName](../../reporting-services/wmi-provider-library-reference/instancename-property-wmi-msreportserver-instance.md)|특정 컴퓨터의 보고서 서버 인스턴스 이름을 반환합니다. 읽기 전용입니다.|  
|[IsSharePointIntegrated](../../reporting-services/wmi-provider-library-reference/issharepointintegrated-property-wmi-msreportserver-instance.md)|보고서 서버가 SharePoint 통합 모드로 구성되어 있는지 여부를 나타냅니다.|  
|[버전](../../reporting-services/wmi-provider-library-reference/version-property-wmi-msreportserver-instance.md)|보고서 서버의 버전을 반환합니다.|
