# <a name="configurationsetting-property---senderemailaddress"></a>SenderEmailAddress ConfigurationSetting 속성-
  보고서 서버에서 전자 메일을 보낼 때 사용하는 주소를 가져옵니다. 읽기 전용입니다.  
  
## <a name="syntax"></a>구문  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## <a name="property-values"></a>속성 값  
 보고서 서버에서 사용하는 메일 주소를 나타내는 읽기 전용 **String** 개체입니다.
