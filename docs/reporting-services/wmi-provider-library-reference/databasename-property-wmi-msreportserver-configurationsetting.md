# DatabaseName 속성(WMI MSReportServer_ConfigurationSetting)
  보고서 서버 데이터베이스를 호스팅하는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 인스턴스의 이름을 지정합니다. 읽기 전용입니다.  
  
## 구문  
  
```vb  
Public Dim DatabaseName As String  
```  
  
```csharp  
public string DatabaseName;  
```  
  
## 속성 값  
 값이 보고서 서버 데이터베이스의 데이터베이스 인스턴스 이름을 나타내는 **String** 개체입니다.
