# SQL Server 2016의 SQL Server Reporting Services에서 중단된 기능
<a id="discontinued-functionality-to-sql-server-reporting-services-in-sql-server-2016" class="xliff"></a>

[!INCLUDE [ssrs-appliesto](../includes/ssrs-appliesto.md)] [!INCLUDE [ssrs-appliesto-2016](../includes/ssrs-appliesto-2016.md)] [!INCLUDE [ssrs-appliesto-not-2017](../includes/ssrs-appliesto-not-2017.md)] [!INCLUDE [ssrs-appliesto-not-pbirs](../includes/ssrs-appliesto-not-pbirs.md)]

[!INCLUDE [ssrs-previous-versions](../includes/ssrs-previous-versions.md)]

이 항목에서는 더 이상 SQL Server 2016에서 사용할 수 있는 SQL Server Reporting Services 기능에 설명 합니다. 특정 버전의 운영 체제 또는 Microsoft 인터넷 정보 서비스 (IIS) 지원 되지 않는 지원에 대 한 공지는 포함 되지 않습니다. 이러한 요구 사항에 대한 자세한 내용은 [SQL Server 2016 설치를 위한 하드웨어 및 소프트웨어 요구 사항](../sql-server/install/hardware-and-software-requirements-for-installing-sql-server.md)을 참조하세요.

*제공이 중단된 기능* 은 더 이상 지원되지 않는 기능입니다. 이러한 기능은 제품에서 실제로 제거되었을 수도 있습니다. 다음 기능이 중단됩니다.

|기능|대체 기능 또는 해결 방법|
|-|-|
|웹 포털을 통해 보고서 모델을 업로드|SOAP API를 통해 계속 수행할 수 있습니다.|
|웹 포털을 통해 보고서 모델 관리|SOAP API를 통해 계속 수행할 수 있습니다.|
