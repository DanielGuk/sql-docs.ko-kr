# <a name="discontinued-functionality-to-sql-server-reporting-services-in-sql-server"></a>SQL Server에서 SQL Server Reporting Services에 지원 되지 않는 기능 

  이 항목에서는 설명 [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] 더 이상 SQL Server 2016에서 사용할 수 있는 기능입니다. 운영 체제 또는 [!INCLUDE[msCoName](../includes/msconame-md.md)] 인터넷 정보 서비스(IIS)의 특정 버전에 대해 더 이상 지원되지 않는 기능에 대해서는 언급하지 않습니다. 이러한 요구 사항에 대한 자세한 내용은 [SQL Server 2016 설치를 위한 하드웨어 및 소프트웨어 요구 사항](../sql-server/install/hardware-and-software-requirements-for-installing-sql-server.md)을 참조하세요.  
  
*제공이 중단된 기능* 은 더 이상 지원되지 않는 기능입니다. 이러한 기능은 제품에서 실제로 제거되었을 수도 있습니다. 다음 기능이 중단됩니다.  
  
|기능|대체 기능 또는 해결 방법|
|-|-|
|다음을 통해 보고서 모델 업로드 [!INCLUDE[ssRSWebPortal](../includes/ssrswebportal.md)]|SOAP API를 통해 계속 수행할 수 있습니다.|  
|다음을 통해 보고서 모델 관리 [!INCLUDE[ssRSWebPortal](../includes/ssrswebportal.md)]|SOAP API를 통해 계속 수행할 수 있습니다.|
