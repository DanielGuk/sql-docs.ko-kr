# <a name="changelog-for-the-report-viewer-controls"></a>보고서 뷰어 컨트롤의 변경 로그

이 변경 로그는 WinForms 및 WebForms 보고서 뷰어 컨트롤의 릴리스를 추적합니다.

## <a name="15900148"></a>15.900.148
사용 업데이트
 - 매개 변수 없는 보고서가 Server.LoadReportDefinition 통해 로드되지 않도록 하는 버그 수정

[WebForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Webforms/150.900.148) 보고서 뷰어 컨트롤
 - RTL 페이지(*direction: rtl;* css 속성을 사용하여 텍스트 흐름을 변경하는 페이지)에 포함하도록 지원합니다.
 - *IReportViewerMessages5* 지역화 인터페이스를 통해 인쇄 대화 상자 텍스트를 사용자 지정하도록 지원합니다.
 - 접근성 지원 개선

[WinForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Winforms/150.900.148) 보고서 뷰어 컨트롤
 - 높은 DPI 모드에서 앱을 실행 중인 경우 인쇄 문제 수정
