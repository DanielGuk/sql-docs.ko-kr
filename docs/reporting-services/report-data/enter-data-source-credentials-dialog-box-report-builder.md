# 데이터 원본 자격 증명 입력 대화 상자(보고서 작성기)
  **데이터 원본 자격 증명 입력** 대화 상자를 사용하여 데이터 원본에서 데이터를 가져오기 위한 자격 증명을 제공할 수 있습니다.  
  
## 옵션  
 **사용자 이름**  
 데이터 원본 로그온에 사용할 사용자 이름을 입력합니다.  
  
 **암호**  
 데이터 원본 로그온에 사용할 암호를 입력합니다.
