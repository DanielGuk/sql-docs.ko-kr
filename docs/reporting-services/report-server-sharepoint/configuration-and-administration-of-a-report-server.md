# <a name="configuration-and-administration-of-a-report-server"></a>구성 및 관리의 보고서 서버

SQL Server Reporting Services는 광범위 한 즉시 사용할 도구 및 서비스 만들기, 배포 및 확장 하 고 보고 기능을 사용자 지정할 수 있는 프로그래밍 기능도 물론 조직에 대 한 보고서를 관리할 수 있도록 제공 하는 서버 기반 보고 플랫폼입니다. 보고 환경을 SharePoint 제품과 통합하면 SharePoint 사이트에서 제공하는 공동 작업 환경을 사용하는 이점을 얻을 수 있습니다.
