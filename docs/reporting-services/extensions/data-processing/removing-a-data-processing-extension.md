# <a name="removing-a-data-processing-extension"></a>데이터 처리 확장 프로그램 제거
  데이터 처리 확장 프로그램을 제거 하려면 제거의 **확장** 구성 파일에서 데이터 처리 확장 프로그램에 대 한 요소입니다. 보고서 서버와 보고서 디자이너에 대 한 항목을 만든 경우 제거 된 **확장** 요소는 RSReportServer.config 파일과 RSReportDesigner.config 파일에서 합니다. 구성 정보를 제거한 후에는 구성 요소에서 데이터 처리 확장 프로그램을 사용할 수 없습니다.
