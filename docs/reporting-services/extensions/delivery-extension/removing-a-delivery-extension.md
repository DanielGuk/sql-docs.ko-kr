# <a name="removing-a-delivery-extension"></a>배달 확장 프로그램 제거
  제거 하는 [!INCLUDE[ssRSnoversion](../../../includes/ssrsnoversion-md.md)] 배달 확장 프로그램 제거는 **확장** 구성 파일에서 배달 확장 프로그램에 대 한 요소입니다. 구성 정보를 제거한 후에는 보고서 서버에서 배달 확장 프로그램을 사용할 수 없습니다.  
  
 배달 확장 프로그램의 해당 되 면 **확장** 보고서 서버에 등록 되어 있지 않은지, 요소가 구성 파일에서 제거 됩니다. 보고서 서버는 배달 확장 프로그램 목록에서 항목을 제거하고 해당 배달 확장 프로그램을 사용하는 구독을 비활성화합니다. 배달 확장 프로그램이 제거되면 사용자가 이 프로그램을 알림 방법으로 선택할 수 없게 됩니다.
