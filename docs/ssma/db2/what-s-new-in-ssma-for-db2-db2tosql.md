# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>기능 &#39; s d b 2 용 SSMA의 새로운 (DB2ToSQL)
이 항목에서는 각 릴리스의 DB2 변경에 대 한 SSMA를 나열 합니다.  

## <a name="may-2016"></a>2016 년 5 월  
D b 2 용 SSMA의 2016 년 5 월 릴리스에서 다음과 같은 변경 내용이 포함 되어 있습니다.  

1.  SQL Server 2016에 대 한 지원 추가
2.  SQL Server 메모리 내 모드 및 hekaton 기능으로 추가 된 DB2 메모리 내 모드와 일반 테이블 변환
3.  SQL Server 정책 개체 (d b 2에 대 한 행 수준 보안)에 추가 된 변환 DB2 액세스 제어의
4.  SQL Server 임시 테이블에 추가 된 DB2 시스템 버전 테이블 변환
5.  향상 된 DB2 파서와 해결 프로그램
6.  .NET 2.0에 대 한 설치 관리자 검사를 제거합니다.
7.  불필요 한 *.dll Db2 설치 관리자에서 제거
8.  프로젝트"저장" 고정 및 SSMA 콘솔에 대 한 프로젝트 열기 명령
9.  SSMA 콘솔에 대 한 고정된 "securepassword" 명령
10. 초기 로드에 대 한 개체의 계산 고정
11. 전역 설정에서 수정 된 버그
  
## <a name="march-2016"></a>2016 년 3 월  
D b 2 용 SSMA의 2016 년 3 월 미리 보기 릴리스에서 다음과 같은 변경을 포함 되어 있습니다.  
  
1.  SQL Server 2016으로 마이그레이션 지원  
  
## <a name="january-2016"></a>2016 년 1 월  
D b 2 용 SSMA의 2016 년 1 월 유지 보수 릴리스는 다음과 같은 변경을 포함 되어 있습니다.  
  
1.  다양 한 표준 함수에 대 한 지원 추가  
  
2.  고정된 DB2 파서 오류  
  
3.  고정된 DB2 v9 zOS 지원 (RFC 5690920)  
  
4.  고정된 DB2 식별자 오류를 확인할 수 없는 변환 하는 동안  
  
5.  SSMA (RFC 5706203)에 추가 된 보기 로그 메뉴 항목  
  
6.  추가 된 원격 분석
