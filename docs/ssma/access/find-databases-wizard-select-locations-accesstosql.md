# <a name="find-databases-wizard-select-locations-accesstosql"></a>데이터베이스 마법사 (선택 위치)를 찾을 (AccessToSQL)
데이터베이스 검색 마법사의 위치 선택 페이지에서 Access 데이터베이스를 찾기 위한 검색 매개 변수를 입력 합니다.  
  
검색 데이터베이스 마법사를 열려면는 **파일** 메뉴를 클릭 하 여 **찾을 데이터베이스**합니다. 이 대화 상자를 사용 하는 방법에 대 한 자세한 내용은 참조 하십시오. [추가 및 제거 Access 데이터베이스 파일](http://msdn.microsoft.com/en-us/e944c740-4c8a-4bc1-b0ed-be57bc06dced)  
  
## <a name="options"></a>옵션  
**찾아보기**  
컴퓨터 또는 네트워크를 찾으려면 여기를 클릭 합니다. 폴더 또는 위치를 클릭 하 고 검색할 선택 **확인**합니다. 클릭 **추가** 목록에이 위치를 추가 합니다.  
  
**경로 상자**  
위치 상자에 추가 하려면 경로 보여 줍니다. 검색을 클릭 한 다음 원하는 경로를 입력 하거나 찾습니다는 **추가** 단추입니다.  
  
**위치 상자**  
SSMA를 검색 하는 위치를 나열 합니다. 클릭는 **추가** 단추를 지정된 된 경로이 상자에 추가 됩니다.  
  
**추가**  
경로 텍스트 상자에 위치 목록에 위치를 추가합니다.  
  
**바꾸기**  
위치 목록에 항목을 선택 하는 경우 항목 경로 텍스트 상자에 선택한 항목을 대체 합니다.  
  
**제거**  
검색할 위치를 목록에서 선택한 항목을 제거 합니다.  
  
**전체 또는 일부 파일 이름**  
이름이 특정 문자열을 포함 하는 데이터베이스를 찾기 위해 문자열을 입력 합니다. 와일드 카드 문자를 사용 하지 마십시오.  
  
**만든 날짜**  
특정 날짜 범위 내에서 작성 된 데이터베이스를 찾기 위해에 시작 날짜를 입력의 **에서** 상자와 끝 날짜는 **를** 상자입니다.  
  
**마지막 업데이트 날짜**  
특정 날짜 범위 내에서 마지막으로 업데이트 된 데이터베이스를 찾기 위해에 시작 날짜를 입력의 **에서** 상자와 끝 날짜는 **를** 상자입니다.  
  
**크기**  
크기 조건과 일치 하는 데이터베이스를 찾으려면 비교 문자 첫 번째 상자에을 크기의 숫자를 입력 선택한 다음 세 번째 상자에서 바이트, 킬로바이트 또는 메가바이트를 선택 합니다.  
  
**소유자**  
소유자의 전체 Windows 사용자 이름을 입력 합니다. 으로 서식을 지정할 수 있습니다이 *도메인*\\*사용자*합니다.
